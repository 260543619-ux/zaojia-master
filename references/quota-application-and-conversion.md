# 定额套用与换算实操

本文件用于辅助造价员进行定额套用思路、换算检查和组价复核。定额编号、消耗量、基价、费用标准必须以项目所在地现行定额库、补充定额、软件导出和地方解释为准；本文件不凭空生成定额编号。

## 一、套定额前置判断

| 判断项 | 要确认的资料 | 影响 |
|---|---|---|
| 工程所在地 | 省/市定额版本、费用文件 | 决定定额库和取费口径 |
| 专业类别 | 建筑、装饰、安装、市政、园林、轨道等 | 决定专业定额册 |
| 清单项目特征 | 材质、规格、强度、厚度、部位、施工方法 | 决定套项和换算 |
| 清单量 vs 定额量 | 清单计算规则、定额计算规则 | 影响综合单价组价 |
| 施工条件 | 运距、高度、泵送、降排水、夜间施工等 | 影响调整系数或措施费 |

## 二、定额套用路径

1. 识别清单项目名称和项目特征。
2. 确定专业定额册和章节。
3. 查找最匹配定额子目；没有完全匹配时，列“类似子目 + 需换算内容”。
4. 判断清单量是否可直接作为定额工程量；不能直接使用时，说明换算路径。
5. 调整人工、材料、机械单价，标注信息价、询价或企业成本来源。
6. 按所在地费用文件计取管理费、利润和风险费。
7. 输出综合单价分析表，并标明所有假设和人工确认点。

## 三、常见换算场景

| 场景 | 换算思路 | 需上传/确认资料 |
|---|---|---|
| 混凝土强度等级变化 | 保留人工、机械消耗量，替换商品砼材料单价差；地方定额另有规定的按规定 | 当地商品砼信息价、定额换算说明 |
| 混凝土泵送/非泵送 | 判断泵送费是否已含在商品砼价、措施费或单独子目中 | 商品砼价格说明、施工方案、定额说明 |
| 模板高度/支撑高度变化 | 按超高支模、层高增加或支撑体系调整规则处理 | 层高、支模高度、地方定额说明 |
| 土方运距变化 | 按定额规定基础运距和每增运距子目换算 | 弃土场/取土点、运输距离、签证 |
| 防水厚度/道数变化 | 按材料消耗量或相应子目调整 | 做法表、材料价格、定额说明 |
| 管道材质/连接方式变化 | 替换管材主材价，检查连接、支架、试压、保温是否另列 | 规格型号、安装定额说明 |
| 电缆规格变化 | 替换主材价，检查敷设方式、桥架/配管、预留量 | 电缆清册、敷设方式、材料价 |

## 四、清单量与定额量区分

| 项目 | 清单量常见口径 | 定额量常见口径 | 检查重点 |
|---|---|---|---|
| 土方 | 设计图示尺寸 | 可能考虑放坡、工作面、运输 | 不得混用在同一列 |
| 钢筋 | 设计净量或规则量 | 可能含损耗、接头、措施筋口径 | 平法规则和软件设置 |
| 模板 | 接触面积 | 按定额规则计算 | 是否与混凝土清单量混同 |
| 脚手架 | 建筑面积或垂直投影 | 按措施规则 | 是否重复签证 |

## 五、输出要求

- 不确定定额编号时，输出“套项思路 + 需用户上传的定额截图/软件导出”。
- 换算必须列明：原子目、换算原因、换算字段、价格来源、人工确认点。
- 不能用全国参考指标替代项目所在地定额消耗量。

## 六、端到端组价示例：C35 商品混凝土矩形柱

> 示例用于展示工作路径，不代表任何地区定额编号或消耗量。实际项目必须以所在地定额库、费用文件、信息价和软件导出为准。

### 1. 清单信息

| 字段 | 示例 |
|---|---|
| 清单编码 | 010502001001 |
| 项目名称 | 矩形柱 |
| 项目特征 | C35 商品混凝土，泵送，层高 4.5m |
| 单位 | m3 |
| 清单工程量 | 100.00 |

### 2. 套项思路

| 工作内容 | 套项/处理思路 | 判断点 |
|---|---|---|
| 柱混凝土浇捣 | 查找现浇混凝土柱相应子目 | 清单量是否可作为定额量 |
| 商品砼 C35 | 若定额基价按 C30，替换商品砼材料价差 | 当地 C30/C35 信息价 |
| 泵送 | 判断泵送费是否含在商品砼价、单独泵送子目或措施费 | 商品砼价格说明、定额说明 |
| 柱模板 | 查找矩形柱模板子目 | 模板工程量需按接触面积另算 |
| 层高 4.5m | 判断是否触发模板超高/支撑高度增加 | 地方定额层高基准和超高规则 |

### 3. 换算逻辑

- **混凝土强度换算**：若原子目材料为 C30 商品砼，且人工、机械消耗不因强度变化调整，则保留人工机械，替换 C35 与 C30 商品砼单价差。
- **泵送费处理**：如果当地信息价 C35 商品砼已含泵送，则不得重复套泵送；如不含，按定额说明或合同约定单列。
- **模板超高处理**：层高 4.5m 是否超出定额基准高度，要看所在地模板章节说明；触发时按超高增加子目或调整系数处理。

### 4. 输出成果

使用 `templates/comprehensive-unit-price-analysis.md` 输出：

1. 清单项基本信息。
2. 定额套用与换算表。
3. 人材机明细。
4. 管理费、利润、风险费和综合单价汇总。
5. 人工确认点：C30/C35 信息价、泵送是否含价、模板超高规则、清单量/模板量口径。
