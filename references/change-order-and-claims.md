# 工程变更、签证与索赔

## 一、工程变更

### 1.1 变更类型

| 变更类型 | 说明 | 谁提出 | 费用承担 |
|---------|------|-------|---------|
| 设计变更 | 设计图纸修改（图纸会审/设计交底/施工过程） | 设计单位/发包人 | 发包人 |
| 发包人变更 | 发包人改变功能/标准/范围/材料/工期等 | 发包人 | 发包人 |
| 承包人建议变更 | 承包人为优化工艺/节约成本/缩短工期提出 | 承包人 | 承包人（节约归己） |
| 不可抗力/外部条件变更 | 地质条件/地下管线/文物等 | — | 发包人 |
| 新增工程 | 原合同范围外的额外工程（GB/T 50500-2024新增概念） | 发包人 | 另行计价 |

### 1.2 GB/T 50500-2024 变更 vs 新增工程的区分

| 维度 | 工程变更 | 新增工程 |
|------|---------|---------|
| 定义 | 合同范围内工程内容的变化 | 合同范围外新增的工程内容 |
| 计价方式 | 按合同约定的变更定价原则 | 双方另行协商或参照市场价格 |
| 合同影响 | 对原合同的调整 | 可签订补充协议 |
| 与原工程关联 | 紧密关联，不可分割 | 可独立于原工程 |

*注：GB/T 50500-2024删除了原2013版中的"工程签证"条款，以更严谨的变更和新增工程规则替代。*

### 1.3 变更定价原则（优先顺序）

**三优先原则：**

1. **合同中已有适用价格** → 直接采用合同综合单价
2. **合同中只有类似价格** → 参照类似综合单价调整（调整主要差异项）
3. **合同中无适用或类似价格** → 双方协商确定
   - 按成本加利润方法：人工+材料+机械+管理费+利润+税金
   - 参照市场价或相关定额重新组价

**操作要点：**
- 相似度判定按项目特征描述比对
- 如仅规格不同（如C30→C35混凝土），可只调整材料费部分
- 新组价的管理费率和利润率应不高于投标报价中同类项目水平

### 1.4 变更程序（时效要求）

```
变更事件发生
  │
  ├── 发包人发出变更指令（书面）
  │
  ├── 承包人在14天内提交变更费用报告
  │      │
  │      ├── 发包人/监理在14天内审核确认
  │      │
  │      └── 逾期未回复视为认可承包人报价
  │
  ├── 双方确认变更费用
  │
  └── 变更费用计入最近一期进度款（或过程结算）
```

**金额归属**：
- 已确认变更费用：计入进度款
- 暂未达成一致的：按发包人暂定价计入进度款（不低于80%），剩余争议部分待解决后支付

## 二、现场签证

### 2.1 签证范围

- 施工图纸和设计变更未包含的，但在施工中实际发生的工作
- 合同清单外的零星用工、零星机械、零星材料
- 返工签证：因发包人原因造成的返工
- 隐蔽工程签证：需现场核实才能计量的工程内容
- 停水停电签证：非承包人原因造成的停窝工损失
- 工期签证：非承包人原因造成的工期延误

### 2.2 签证"四要素"

| 要素 | 要求 |
|------|------|
| **及时性** | 事件发生后7-14天内申报，超期视为放弃（合同约定） |
| **准确性** | 明确内容、规格、数量、时间（精确到日） |
| **完整性** | 签证单+计算书+草图/照片+指令文件（证据链完整） |
| **合法性** | 有权签发人签署（合同约定的发包人代表/监理） |

### 2.3 签证常见问题

- **事后补签**：隐蔽工程覆盖后不可复核，必须在覆盖前签证
- **签证不明确**：仅写"按实计"而无实际数据，结算时无法计量
- **无权签字**：合同约定的发包人代表或总监以外人员签署的签证无效
- **重复签证**：同一工作已在清单中计取，又签证一次
- **签证过期**：未在合同约定时效内提交，发包人可拒绝

## 三、工程索赔

### 3.1 索赔类型

| 索赔类型 | 索赔内容 | 常见情形 |
|---------|---------|---------|
| 工期索赔 | 只延长工期 | 发包人图纸延误、甲供材迟交 |
| 费用索赔 | 只补偿费用 | 发包人原因导致窝工（不含工期影响） |
| 工期+费用索赔 | 既顺延工期又补偿费用 | 设计变更导致工程量大幅增加、不可抗力 |
| 利润索赔 | 要求补偿利润损失 | 发包人解除合同、发包人指示错误 |

### 3.2 索赔费用组成

**可索赔费用：**
- 人工费：窝工费（一般按日工资的50-70%）、额外用工费
- 材料费：材料积压仓储费、材料涨价（超出约定幅度的部分）
- 机械费：停置费（自有机械按折旧费、租赁机械按租赁合同价）
- 管理费：现场管理费（窝工期间必要管理人员费用）
- 周转材料：停滞期间的租赁费
- 保函手续费：工期延长导致保函费用增加
- 利息：发包人延迟支付结算款的利息

**不可索赔费用：**
- 承包人自身管理不善导致的损失
- 承包人自身施工组织不当的成本
- 承包人可投保而未投保的意外损失
- 合同明确排除的费用

### 3.3 索赔时效

| 步骤 | 时效 | 超时后果 |
|------|------|---------|
| 索赔意向通知 | 索赔事件发生后28天内 | 丧失索赔权 |
| 正式索赔报告 | 索赔意向通知后28天内 | 丧失索赔权 |
| 发包人审核回复 | 收到索赔报告后28天内 | 视为认可 |
| 证据补充 | 发包人要求后14天内 | — |

### 3.4 索赔证据链

完整的索赔应提供：
1. 索赔信（说明索赔事项和金额）
2. 合同依据（引用具体条款）
3. 事实证据（现场照片/监理日志/会议纪要/往来函件/记录天气）
4. 费用计算书（计算方法+定额/市场价格依据）
5. 工期影响分析（横道图/网络计划图，关键线路分析）

## 四、反索赔（发包人向承包人索赔）

发包人可提出的反索赔情形：
- 工期延误违约金（按合同约定的违约金标准和上限）
- 质量不合格的返工费用（承包人承担）
- 承包人原因导致的第三方损失
- 承包人擅自使用不合格材料
- 质保期内修复费用（承包人原因造成的缺陷）
- 承包人原因解除合同的损失（重新招标增加费用）

## 五、变更签证管理最佳实践

1. **一单一签**：一个变更签证对应一份变更指令（变更号唯一）
2. **编号管理**：台账统一编号，方便追溯
3. **影像资料**：隐蔽工程等重点签证务必附带照片/视频
4. **分层审批**：按金额分级审批（X万以下现场代表，X-Y万项目经理，Y万以上公司审批）
5. **避免口头指令**：发包人所有变更指令必须书面发出
6. **合同模板优化**：在合同中明确变更定价的原则和时效
