# 合同类型、风险分配与价款调整

## 一、合同类型选择

### 1.1 GB/T 50500-2024 三种合同类型

| 合同类型 | 工程量风险 | 单价/总价风险 | 适用条件 |
|---------|----------|-------------|---------|
| 单价合同 | 发包人 | 承包人 | 工程量不确定，设计深度不足（初设阶段招标） |
| 总价合同 | 承包人 | 承包人 | 图纸完整，工程量可准确计量（施工图阶段招标） |
| 成本加酬金 | 发包人 | 发包人 | 抢险救灾、新技术、紧急工程 |

### 1.2 选择决策矩阵

| 工程条件 | → 推荐合同类型 |
|---------|--------------|
| 施工图完整 + 规模≤中型 + 工期≤12月 | 总价合同 |
| 初步设计深度招标 + 地下条件不确定 | 单价合同 |
| 抢修/抢险/特殊工艺/工期极端紧张 | 成本加酬金 |
| 市场价格波动剧烈（通胀时期） | 单价合同+调价条款 |
| 固定总价合同 + 工期>18月 | 不推荐，风险过高 |

### 1.3 风险分配"红线"

GB/T 50500-2024 明确**禁止**：
- 合同条款中出现"所有风险由承包人承担"
- 要求承包人承担无法预见的风险
- 综合单价中一次包死，排除所有调价可能

## 二、综合单价风险范围

### 2.1 风险分摊原则

| 风险类型 | 发包人承担 | 承包人承担 |
|---------|----------|----------|
| 法律变化（新税种/新规费） | ✓ | |
| 物价波动（超出约定幅度） | ✓（超出部分） | ✓（约定幅度内） |
| 工程量变化（单价合同） | ✓ | |
| 工程量变化（总价合同） | | ✓ |
| 施工条件变化（不可预见地质等） | ✓ | |
| 施工方案和施工组织 | | ✓ |
| 管理效率和技术水平 | | ✓ |
| 工程量清单缺陷（单价合同） | ✓ | |
| 工程量清单缺陷（总价合同） | | ✓ |
| 安全生产措施费调整 | 按实调整 | |

## 三、价款调整公式与方法

### 3.1 物价波动调价（GB/T 50500-2024 附录A）

#### 方法一：价格指数调差法

```
ΔP = P₀ × [A + (B₁×Ft₁/F₀₁ + B₂×Ft₂/F₀₂ + ... + Bn×Ftn/F₀n) - 1]
```

其中：
- ΔP — 调价金额
- P₀ — 合同价款中已完成工程量的金额
- A — 固定系数（不可调部分，一般0.15-0.35）
- B₁...Bn — 各可调因子权重系数（A + B₁+...+Bn = 1）
- Ft₁...Ftn — 各可调因子现行价格指数
- F₀₁...F₀n — 各可调因子基本价格指数

#### 方法二：价格信息调差法

适用于主要材料（钢筋、混凝土、电缆等）单品种调差：

```
ΔP = Q × (Pt - P₀) × (1 + 税率)
```

其中：
- Q — 实际用量（按合同约定计量规则）
- Pt — 施工期信息价/市场价
- P₀ — 基准期信息价/市场价

### 3.2 风险幅度约定

| 材料类型 | 承包人承担幅度 | 超出部分承担 |
|---------|-------------|------------|
| 主要材料（钢筋、混凝土等） | ±5%以内 | 超出部分发包人承担 |
| 次要材料 | ±10%以内 | 超出部分发包人承担 |
| 机械台班 | ±10%以内 | 协商或参考造价信息 |

### 3.3 工程量变化调价

工程量变化超过清单量±15%时（单价合同）：

1. **增加 > 15%**：超出15%以上部分的综合单价可调低
2. **减少 > 15%**：剩余部分的综合单价可调高
3. **调整理由**：工程量的重大变化会影响承包人的固定成本摊销量

## 四、不可抗力费用分摊

| 费用类别 | 承担方 |
|---------|-------|
| 工程本身损害 | 发包人 |
| 第三方人员伤亡和财产损失 | 发包人 |
| 承包人人员伤亡 | 承包人 |
| 承包人机械设备损坏 | 承包人 |
| 停工期间必要的管理人员和保卫人员费用 | 发包人 |
| 清理修复费用 | 发包人 |
| 工期延误 | 发包人确认顺延 |

---

## 五、工期延误的费用处理

| 延误原因 | 工期 | 费用 |
|---------|------|------|
| 发包人原因（图纸迟交/甲供材晚到等） | 顺延 | 发包人承担 |
| 承包人原因（组织不当/劳力不足等） | 不顺延 | 承包人承担 |
| 不可抗力 | 顺延 | 按四节分摊 |
| 异常恶劣天气 | 顺延 | 发包人承担 |

---

## 六、缺陷责任与质量保证金

- 缺陷责任期：一般为1-2年（最长不超过2年）
- 质量保证金：不超过工程价款结算总额的**3%**
- 缺陷责任期满后14天内，发包人退还剩余质保金
- 缺陷责任期内因承包人原因造成的缺陷，修复费用从质保金中扣除；非承包人原因的，另行支付

---

## 七、合同价款条款审查清单

签订合同前，造价条款应至少涵盖：

- [ ] 合同类型明确（单价/总价/成本加酬金）
- [ ] 工程量清单完整性责任归属
- [ ] 预付款比例及扣回方式
- [ ] 进度款支付比例及审核时限
- [ ] 变更定价原则（三优先顺序）
- [ ] 物价波动调价方法及风险幅度
- [ ] 工程量变化超出±15%的处置
- [ ] 暂估价确认流程
- [ ] 过程结算节点划分
- [ ] 竣工结算时限
- [ ] 质量保证金比例（≤3%）及退还条件
- [ ] 争议解决方式
- [ ] 无"无限风险由承包人承担"等违规条款
