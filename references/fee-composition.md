# 建筑安装工程费用组成详解

## 一、费用四级结构（GB/T 50500-2024）

```
建筑安装工程费
  │
  ├── 第一级：分部分项工程费
  │     └── ∑(分部分项工程量 × 综合单价)
  │
  ├── 第二级：措施项目费
  │     ├── 安全生产措施费（不可竞争）
  │     └── 其他措施项目费（可竞争）
  │
  ├── 第三级：其他项目费
  │     ├── 暂列金额
  │     ├── 暂估价
  │     ├── 计日工
  │     └── 总承包服务费
  │
  └── 第四级：税金
        └── 增值税（单独计列）
```

*注：GB/T 50500-2024 将原2013版中的"规费"拆分并入：社会保险费、住房公积金并入人工费；工会经费、职工教育经费等并入企业管理费。不再作为独立费用项单独列项。*

## 二、综合单价拆解

### 综合单价 = 人工费 + 材料费 + 施工机具使用费 + 企业管理费 + 利润 + 风险费

### 2.1 人工费（新标准扩大口径）

GB/T 50500-2024 的人工费包含：
- 计时/计件工资
- 奖金
- 津贴补贴
- 加班加点工资
- 特殊情况下支付的工资
- **社会保险费**（原规费项，已并入）
- **住房公积金**（原规费项，已并入）
- 工会经费
- 职工教育经费
- 劳动保护费
- 职工福利费

### 2.2 材料费

| 构成 | 说明 |
|------|------|
| 材料原价 | 出厂价或市场供应价 |
| 运杂费 | 从来源地到工地仓库的运输、装卸等费用 |
| 运输损耗费 | 运输途中的合理损耗 |
| 采购及保管费 | 采购、验收、仓储、保管费用(一般2-2.5%) |
| 检验试验费 | 一般鉴定、检查费用（不含新结构/新材料试验费） |

### 2.3 施工机具使用费

- 折旧费
- 大修理费
- 经常修理费
- 安拆费及场外运费
- 人工费（机上操作人员）
- 燃料动力费
- 税费（车船使用税等）

### 2.4 企业管理费

**取费基数**（各省不同）：
- 以人工费为基数（广东、四川等）
- 以人工费+机械费为基数（北京、上海、浙江等）

**取费比例**：一般占分部分项工程费的3-8%（房建），5-10%（市政）

**包含内容**：
- 管理人员工资
- 办公费
- 差旅交通费
- 固定资产使用费
- 工具用具使用费
- 劳动保险和福利费
- 劳动保护费
- 工会经费和职工教育经费
- 财产保险费
- 财务费（利息支出等）
- 税金（房产税/车船税等，不含增值税）
- 其他（审计费/法律顾问费/业务招待费等）

### 2.5 利润

- 取费基数：同管理费取费基数
- 一般利润率：5-7%（房建），7-10%（市政/安装）

### 2.6 风险费

- 材料价格波动（合同约定范围内由承包人承担的部分）
- 施工条件变化风险（可预见的常规风险）
- 不包括：法规变化、不可抗力、发包人指令变更

---

## 三、措施项目费详解

### 3.1 安全生产措施费（不可竞争）

| 子项 | 计取方式 | 费率范围 |
|------|---------|---------|
| 环境保护费 | 分部分项费×费率 | 0.25-0.5% |
| 文明施工费 | 分部分项费×费率 | 0.5-1.5% |
| 安全施工费 | 分部分项费×费率 | 1.0-2.5% |
| 临时设施费 | 分部分项费×费率 | 0.8-2.5% |
| **合计** | | **约2.5-7%** |

*具体费率以省级主管部门最新文件为准。*

### 3.2 其他措施项目费（可竞争）

| 项目 | 计算方式 | 是否包干 |
|------|---------|---------|
| 脚手架工程费 | 按面积或按项 | 一般包干 |
| 混凝土模板及支架 | 按混凝土量或按项 | 一般包干 |
| 垂直运输费 | 按建筑面积或工期 | 一般包干 |
| 大型机械进出场及安拆 | 按台次 | 可调 |
| 施工排水降水 | 按实计或按项包干 | 视约定 |
| 已完工程保护费 | 按项 | 一般包干 |
| 高层建筑增加费 | 按面积或按百分比 | 一般包干 |
| 冬雨季施工增加费 | 分部分项费×费率 | 一般包干 |
| 夜间施工增加费 | 按夜间施工天数 | 可调 |

## 四、其他项目费详解

### 4.1 暂列金额

| 设计阶段 | 建议比例 |
|---------|---------|
| 初步设计阶段 | 分部分项工程费的10-15% |
| 施工图设计阶段 | 分部分项工程费的5-10% |

**结算处理**：暂列金额余额归发包人所有，如有不足由发包人另行追加。

### 4.2 暂估价

| 类型 | 列项方式 | 结算处理 |
|------|---------|---------|
| 材料暂估价 | 在分部分项清单中直接标注暂估单价 | 确认单价后调整综合单价 |
| 设备暂估价 | 同上 | 同上 |
| 专业工程暂估价 | 在其他项目清单中列出暂估总价 | 确认后调整合同价 |

### 4.3 计日工

- 人工：按暂定数量×投标人报的计日工单价
- 材料：按暂定数量×投标人报的计日工材料单价
- 机械：按暂定数量×投标人报的计日工机械单价
- 结算：按发包人实际签证确认的数量

### 4.4 总承包服务费

**服务范围**：
- 对发包人直接发包的专业工程进行总承包管理和协调
- 对甲供材进行保管和配合（如需承包人保管，则计取保管费）

**计价方式**：
- 总价方式：按服务内容报价
- 费率方式：按分包工程造价的百分比（一般1-3%）

## 五、税金（增值税）

### 5.1 建筑业增值税

| 计税方式 | 适用情况 | 税率 |
|---------|---------|------|
| 一般计税 | 一般纳税人 | 9%（销项税，进项税可抵扣） |
| 简易计税 | 小规模纳税人/清包工/甲供工程/老项目 | 3%（全额征收，不可抵扣） |

### 5.2 GB/T 50500-2024 增值税处理

- **综合单价不含增值税**
- 在单位工程费汇总表中单独列示"增值税"
- 增值税 = (分部分项费 + 措施项目费 + 其他项目费) × 增值税税率
- 附加税（城建税、教育费附加、地方教育附加）已并入企业管理费

---

## 六、不可竞争费用汇总

以下费用投标人**不得下浮、不得优惠**：

| 不可竞争费 | 依据 |
|----------|------|
| 安全生产措施费（原安全文明施工费） | 国家安全生产法规，省级主管部门核定费率 |
| 增值税 | 税法，按国家规定税率全额计取 |

*注1：GB/T 50500-2024 将原2013版"规费"（社会保险费、住房公积金）并入人工费/企业管理费，不再作为独立费用项。*
*注2：2024标准中更名为"安全生产措施费"（术语2.0.4），实务中与旧称"安全文明施工费"常混用。*
