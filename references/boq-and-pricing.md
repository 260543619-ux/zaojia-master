# 工程量清单编制与综合单价分析

## 一、工程量清单编制

### 1.1 清单框架（GB/T 50500-2024 "5改4"）

```
单位工程费汇总表
  ├── 分部分项工程量清单计价表
  ├── 措施项目清单计价表
  │     ├── 安全生产措施费（不可竞争）
  │     └── 其他措施项目费（可竞争）
  ├── 其他项目清单计价表
  │     ├── 暂列金额
  │     ├── 暂估价
  │     ├── 计日工
  │     └── 总承包服务费
  └── 增值税计价表（单独计列）
```

*注：原2013版中的规费已并入人工费/管理费，不再单独列项。*

### 1.2 项目编码规则（12位）

```
XX - XX - XX - XXX - XXX
专业  分部  分项   顺序  特征码
```

专业代码：
- 01 — 房屋建筑与装饰工程
- 02 — 仿古建筑工程
- 03 — 通用安装工程
- 04 — 市政工程
- 05 — 园林绿化工程
- 06 — 矿山工程
- 07 — 构筑物工程
- 08 — 城市轨道交通工程
- 09 — 爆破工程

### 1.3 分部分项工程量清单编制要点

**五要素（缺一不可）：**
1. **项目编码**：12位，同一招标工程不得重码
2. **项目名称**：按各专业附录中的项目名称，结合工程实际
3. **项目特征描述**：最关键要素！描述不清是合同纠纷第一来源
4. **计量单位**：按各专业附录规定（m、㎡、m³、t、个、套等）
5. **工程量**：按各专业附录计算规则计算

**项目特征描述原则：**
- 设计图纸注明的规格、尺寸、材质必须描述
- 影响综合单价确定的特征必须描述（如混凝土强度等级、钢筋规格）
- 施工工艺和方法一般不描述（由投标人自行确定）
- 无法准确描述的，可注明"详见设计图纸"

**工程量计算精度：**
- 以"t"为单位：保留3位小数
- 以"m³、㎡、m"为单位：保留2位小数
- 以"个、套、台"为单位：取整数

### 1.4 措施项目清单

**安全生产措施费（不可竞争费）：**
| 子项 | 说明 |
|------|------|
| 环境保护费 | 扬尘控制、噪声控制等 |
| 文明施工费 | 现场围挡、场地硬化等 |
| 安全施工费 | 安全防护设施、安全网等 |
| 临时设施费 | 办公、生活、生产临时用房等 |

**其他措施项目费（可竞争）：**
- 脚手架工程费
- 混凝土模板及支架费
- 垂直运输费
- 大型机械进出场及安拆费
- 施工排水降水费
- 已完工程及设备保护费
- 高层建筑施工增加费
- 冬雨季施工增加费
- 夜间施工增加费

### 1.5 其他项目清单

| 项目 | 确定方式 | 结算处理 |
|------|---------|---------|
| 暂列金额 | 招标人按分部分项工程费的10-15%（初设阶段）或5-10%（施工图阶段）估算 | 余额归发包人 |
| 暂估价（材料/设备） | 招标人给定暂估单价和数量 | 确认后调整合同价 |
| 暂估价（专业工程） | 招标人给定暂估总价 | 确认后调整合同价 |
| 计日工 | 招标人给出暂定数量（人工工日、材料数量、机械台班） | 按实际签证确认 |
| 总承包服务费 | 投标人自主报价 | 按合同约定 |

## 二、综合单价组价

### 2.1 综合单价定义（GB/T 50500-2024）

综合单价（不含增值税）= **人工费 + 材料费 + 施工机具使用费 + 企业管理费 + 利润 + 一定范围内风险费**

与2013版关键差异：**不含增值税**，税金在汇总表单独列示。

### 2.2 综合单价组价方法

#### 方法一：定额组价法（最常用）

1. 确定清单项目对应的定额子目
2. 计算每个定额子目的工程量和定额基价
3. 计算人工费：∑(定额人工消耗量 × 人工单价)
4. 计算材料费：∑(定额材料消耗量 × 材料单价)
5. 计算机械费：∑(定额机械台班消耗量 × 机械台班单价)
6. 计算管理费：人工费（或人工+机械费）× 管理费率
7. 计算利润：人工费（或人工+机械费）× 利润率
8. 合计 = 人工+材料+机械+管理+利润+风险

#### 方法二：市场询价法（2024新标准鼓励）

- 适用于有明确市场价格的材料和设备
- 需要至少3个以上市场来源作为支撑
- 应记录询价来源、时间、报价单位

#### 方法三：企业定额法（大型施工企业）

- 基于企业自身历史成本数据
- 更能反映企业真实成本水平
- 是2024新标准推荐的方向

## 三、不平衡报价识别（清标重点）

### 3.1 常见不平衡报价手法

| 手法 | 表现 | 承包人目的 |
|------|------|----------|
| 前置高价法 | 先施工项目单价高，后施工项目单价低 | 加速资金回收 |
| 工程量预期法 | 预计增加量的项目单价高，减少的单价低 | 获取超额利润 |
| 设计不明确法 | 设计模糊项目报高价，待变更后按高价结算 | 利用变更获利 |

### 3.2 识别指标

**单价偏离度** = (投标单价 - 评标基准单价) / 评标基准单价 × 100%

- 偏离度 > 15-20%（具体阈值合同约定）：应要求投标人书面说明
- 偏离度 > 30%：可能存在不合理报价
- 合价偏离金额较大（如超过合同总价1%）的单项：重点审查

### 3.3 GB/T 50500-2024 清标处理规则

根据 GB/T 50500-2024 第6章规定，清标的一般流程（具体以招标文件约定的评标办法为准）：

1. 评标委员会可在推荐中标候选人前，要求投标人对报价进行澄清或说明
2. 澄清范围通常包括：算术误差、细微偏差、报价合理性、报价完整性
3. 澄清不得改变实质性内容（工程范围、工期要求、合同条件、投标总价）
4. 澄清文件经投标人书面确认后，可构成合同文件组成部分
5. 对明显不合理且无合理解释的报价，由评标委员会按招标文件规定的评标办法处理

*注：上述为GB/T 50500-2024的原则性规定，具体处置标准、偏离阈值、处理程序以项目招标文件中的评标办法为准。本Skill不替代招标文件或评标委员会的判断。*

## 四、实务常见问题

1. **项目特征描述不足**：如只写"C30混凝土"，未注明是商品砼还是现场搅拌，导致投标报价差异大
2. **清单漏项**：单价合同下，清单漏项由发包人承担费用
3. **计算规则不统一**：如土方工程，清单按设计图示尺寸计算（不含工作面和放坡），定额按实际开挖量计算
4. **暂列金额比例不当**：初设阶段建议10-15%，施工图阶段5-10%；边境/特殊项目可适当提高
5. **综合单价套用**：结算时不能随意套用投标时的"类似"综合单价，须经双方确认
