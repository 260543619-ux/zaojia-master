# 新旧计价标准混用检查

GB/T 50500-2024 于 2025-09-01 实施。过渡期项目中常见 GB 50500-2013、GB/T 50500-2024、地方旧定额库、软件旧模板和合同专用条款混用，容易造成费用口径、税金、规费、安全生产措施费和综合单价口径错误。

## 一、必须确认的信息

| 信息 | 资料来源 | 影响 |
|---|---|---|
| 合同签订日期 | 合同协议书 | 判断合同适用标准 |
| 招标文件发布日期 | 招标文件 | 判断清单编制依据 |
| 合同约定标准 | 专用条款/计价条款 | 优先级高于通用理解 |
| 软件计价模板 | GCCP/计价软件项目设置 | 决定表格和费用项 |
| 地方执行文件 | 省/市住建部门通知 | 决定当地过渡安排 |

## 二、混用风险清单

| 风险 | 表现 | 检查方式 |
|---|---|---|
| 费用结构混用 | 2024 项目仍单列规费，或 2013 项目误删规费 | 对照合同约定和软件模板 |
| 增值税口径混用 | 综合单价含税/不含税不清 | 检查综合单价分析表和汇总表 |
| 安全文明/安全生产名称混用 | 表名旧称新称混杂 | 判断是否只是软件旧称，还是费用重复 |
| 招标控制价/最高投标限价混用 | 文件标题和条款名称不一致 | 以招标文件和地方规定为准 |
| 甲供材处理混用 | 进入综合单价后扣除或不进入综合单价 | 检查合同和清单说明 |
| 清标规则混用 | 采用新标准清标但招标文件未约定程序 | 检查评标办法 |

## 三、处理原则

1. 先看合同、招标文件和地方执行文件，不直接按日期机械判断。
2. 同一项目不同标段可能适用不同标准，应分标段列示。
3. 软件导出表格使用旧名称时，不一定代表计价口径错误；需核对费用内容。
4. 发现混用时，输出“风险提示 + 需确认资料 + 可能影响”，不要直接改价。

|---|---|---|---|---|---|---|
| 1 |  |  | 费用结构/税金/合同风险 |  | 补充确认/调整表格/保留旧口径 | 疑似/确定/待补 |
