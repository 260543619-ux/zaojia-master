# 成本管理与偏差分析

本文件用于目标成本、动态成本、合同成本、变更成本和结算成本的管理型分析。它面向成本负责人和项目商务管理，不替代财务核算或企业 ERP 数据。

## 一、成本口径

| 口径 | 说明 | 常见用途 |
|---|---|---|
| 目标成本 | 项目立项或方案阶段确定的成本控制目标 | 投资决策、限额设计 |
| 合同成本 | 已签合同金额及合同范围 | 合约规划、付款控制 |
| 动态成本 | 合同成本 + 已发生变更 + 预计变更 + 风险预留 | 月度成本管控 |
| 结算成本 | 已审核或定案的最终工程成本 | 结算管理、复盘 |
| 责任成本 | 按部门或责任主体分解的控制目标 | 内部考核 |

## 二、成本测算输入

- 项目类型、所在地、建筑面积、地上/地下规模、结构形式、装配率、装修标准。
- 目标成本指标、类似项目指标、当地造价信息、市场询价。
- 合约规划、已签合同、待签合同、变更签证台账、付款台账。

## 三、动态成本计算框架

```text
动态成本 = 已签合同金额
        + 待签合同预测金额
        + 已确认变更签证
        + 已发生未确认变更签证
        + 预计风险金额
        - 可确认节余金额
```

## 四、偏差分析

| 偏差类型 | 判断方式 | 输出建议 |
|---|---|---|
| 目标偏差 | 动态成本 - 目标成本 | 说明超支/节余金额和比例 |
| 合同偏差 | 已签合同 - 合约规划 | 判断招采结果是否偏离计划 |
| 变更偏差 | 变更签证金额 / 合同金额 | 识别设计、现场、业主需求变化 |
| 付款偏差 | 已付款 / 已完产值 | 判断超付、欠付或付款滞后 |
| 指标偏差 | 单方造价 vs 类似项目/目标指标 | 仅作管理预警，不作结算依据 |

## 五、管理层输出

成本负责人通常需要的是“风险排序 + 金额影响 + 下一步动作”，而不是长篇规范解释。输出时优先使用：

- `templates/management-summary.md`
- 合同/变更/付款/风险四类金额汇总表
- 高风险事项清单
- 待决策事项清单

## 六、风险控制

- 目标成本和动态成本必须说明数据口径日期。
- 类似项目指标只能作对标，不能替代当前项目合同和市场价格。
- 预计变更和风险预留必须与已确认金额分列。
- 没有合约规划或合同台账时，只能做初步测算。
