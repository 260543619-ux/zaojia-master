## 八、国际常用术语（供对比）

| English | 中文近似 |
|---------|---------|
| Quantity Surveyor (QS) | 工料测量师 / 造价工程师 |
| Bill of Quantities (BOQ) | 工程量清单 |
| Provisional Sum | 暂列金额（近似但不完全相同） |
| Prime Cost Sum (PC Sum) | 暂估价（近似） |
| Variation / Change Order | 工程变更 |
| Contingency | 不可预见费 / 预备费 |
| Preliminaries | 措施项目费（近似） |
| Daywork | 计日工 |
| Nominated Sub-Contractor | 指定分包商 |
| Fluctuation Clause | 物价波动调价条款 |
| Retention | 质量保证金 |
| Practical Completion | 实际竣工（相当于竣工验收） |
| Defects Liability Period | 缺陷责任期 |
| Extension of Time (EOT) | 工期顺延 |
| Letter of Acceptance | 中标通知书 |
| Performance Bond | 履约保函 / 履约保证金 |
| Snagging / Punch List | 缺陷整改清单 |
| Cost Reimbursement Contract | 成本加酬金合同 |
| Design and Build (D&B) | 设计施工总承包 |

---

*注：中英造价术语并非完全一一对应，因为两个法系和计价体系有根本差异。上表中的英文对应仅为理解和沟通之用，在国际合同谈判中应严格按合同定义的术语使用。*
