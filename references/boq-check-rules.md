# 工程量清单检查规则

本文件用于清单特征描述检查、清单漏项检查和清单异常识别。没有完整图纸、设计说明、招标范围和专业界面时，只能输出“疑似问题”，不得认定完整漏项或直接定价。

## 一、清单五要素检查

| 要素 | 检查点 | 常见异常 |
|---|---|---|
| 项目编码 | 是否为空、位数是否正确、同一单位工程是否重复 | 缺编码、重复编码、专业编码不匹配 |
| 项目名称 | 是否能反映工程内容 | 名称过泛、与项目特征不一致 |
| 项目特征 | 是否包含影响组价和计量的关键字段 | 规格、材质、厚度、强度、部位缺失 |
| 计量单位 | 是否符合清单规则和专业惯例 | 单位错用、同类项单位不一致 |
| 工程量 | 是否有数量、精度是否合理 | 空值、负数、异常小数位、量级异常 |

## 二、常见项目特征必查字段

| 专业/项目 | 必查字段 |
|---|---|
| 土方 | 土类别、挖填方式、运输距离、弃土/回填、地下水条件 |
| 混凝土 | 强度等级、构件部位、浇筑方式、商品砼/现场搅拌、抗渗/抗冻要求 |
| 钢筋 | 钢筋种类、规格范围、连接方式、抗震等级、是否含措施筋 |
| 砌体 | 材料种类、规格、强度等级、砂浆强度、墙厚 |
| 防水 | 材料类型、厚度、道数、部位、附加层、保护层 |
| 门窗 | 材质、开启方式、玻璃配置、五金、洞口尺寸、耐火/节能要求 |
| 装饰面层 | 基层、面层材料、厚度、规格、做法、部位 |
| 给排水管 | 材质、规格、连接方式、压力等级、保温/防腐 |
| 电缆电线 | 型号、规格、敷设方式、防火等级、是否含桥架/配管 |
| 风管 | 材质、厚度、规格、保温、防火、连接方式 |
| 消防管道 | 管材、规格、连接方式、压力等级、刷油防腐、保温、试压冲洗 |
| 消防设备 | 设备名称、规格型号、安装方式、控制方式、调试要求 |
| 弱电智能化 | 系统类别、线缆型号、敷设方式、设备参数、调试联动 |
| 暖通设备 | 设备型号、风量/冷量/热量、安装方式、减振、调试 |
| 通风空调水管 | 材质、规格、连接方式、保温厚度、防腐、试压 |
| 市政道路 | 道路结构层、厚度、材料、压实度、基层/面层范围 |
| 市政排水管网 | 管材、管径、接口、基础、埋深、检查井、闭水试验 |
| 市政桥梁 | 桩基、承台、墩台、梁板、支座、伸缩缝、防撞护栏 |
| 市政照明/交通 | 灯杆高度、灯具参数、基础、管线、控制箱、标志标线做法 |

## 三、疑似漏项检查路径

1. 按招标范围确认本次清单应覆盖的专业和界面。
2. 按图纸目录和设计说明列出主要系统、构件、材料和做法。
3. 对照清单分部分项、措施项目、其他项目和暂估价范围。
4. 将未找到对应清单项的内容列入疑似漏项。
5. 对可能包含在综合单价、措施费、暂估价、专业分包或甲供范围内的内容，标记为“需人工确认”。

## 四、常见疑似漏项

| 类型 | 常见漏项 |
|---|---|
| 土建 | 垫层、防水保护层、止水钢板、后浇带、预埋件、洞口加固 |
| 装饰 | 基层处理、找平层、收边收口、踢脚、门窗套、成品保护 |
| 安装 | 套管、支吊架、阀门、保温、防火封堵、系统调试 |
| 市政 | 井室、管道基础、回填、接口处理、交通导改、临时排水 |
| 措施 | 脚手架、模板支撑、垂直运输、降排水、大型机械进出场 |

## 五、安装专业专项检查

| 专业 | 高频问题 | 检查建议 |
|---|---|---|
| 消防 | 管道试压冲洗、刷油防腐、支吊架、系统调试漏列 | 对照系统图和消防设计说明 |
| 给排水 | 套管、支架、阀门、保温、防腐、通水试验漏列 | 对照管材表、系统图和卫生器具表 |
| 电气 | 配管配线、桥架、接线盒、接地、防火封堵漏列 | 对照系统图、平面图、配电箱系统 |
| 弱电 | 设备箱、模块、跳线、调试、联动漏列 | 对照点表和系统说明 |
| 暖通 | 风管保温、软接、阀部件、支吊架、系统调试漏列 | 对照风管平面和设备表 |

## 六、市政专业专项检查

| 专业 | 高频问题 | 检查建议 |
|---|---|---|
| 道路 | 路床整形、基层、透层/粘层、井盖调整、标线漏列 | 对照道路结构图和交通工程图 |
| 排水 | 管道基础、砂垫层、回填、检查井、闭水试验漏列 | 对照纵断面、管道基础图 |
| 给水/燃气/热力 | 阀门井、支墩、冲洗消毒、探伤、防腐漏列 | 对照管线设计说明 |
| 桥梁 | 支座、伸缩缝、桥面铺装、防撞护栏、泄水管漏列 | 对照桥梁总体和构造图 |
| 交通导改 | 围挡、临时道路、导行标志、夜间警示漏列 | 对照施工组织和招标范围 |

## 七、输出要求

- 使用 `templates/boq-feature-check-table.md` 输出特征问题。
- 使用 `templates/boq-missing-items-register.md` 输出疑似漏项。
- 每条问题都要写依据来源：清单行号、设计说明页码、图号或招标范围条款。
- 不输出确定金额；如用户要求估算，必须单独列假设和依据。
