# 工程量清单检查规则

本文件用于清单特征描述检查、清单漏项检查和清单异常识别。没有完整图纸、设计说明、招标范围和专业界面时，只能输出“疑似问题”，不得认定完整漏项或直接定价。

## 一、清单五要素检查

| 要素 | 检查点 | 常见异常 |
|---|---|---|
| 项目编码 | 是否为空、位数是否正确、同一单位工程是否重复 | 缺编码、重复编码、专业编码不匹配 |
| 项目名称 | 是否能反映工程内容 | 名称过泛、与项目特征不一致 |
| 项目特征 | 是否包含影响组价和计量的关键字段 | 规格、材质、厚度、强度、部位缺失 |
| 计量单位 | 是否符合清单规则和专业惯例 | 单位错用、同类项单位不一致 |
| 工程量 | 是否有数量、精度是否合理 | 空值、负数、异常小数位、量级异常 |

## 三、疑似漏项检查路径

1. 按招标范围确认本次清单应覆盖的专业和界面。
2. 按图纸目录和设计说明列出主要系统、构件、材料和做法。
3. 对照清单分部分项、措施项目、其他项目和暂估价范围。
4. 将未找到对应清单项的内容列入疑似漏项。
5. 对可能包含在综合单价、措施费、暂估价、专业分包或甲供范围内的内容，标记为“需人工确认”。

## 八、量价交叉验证规则

仅检查格式（编码位数、特征是否为空）对用户实际帮助有限。以下规则用于从量价合理性角度发现异常。

### 8.1 同专业内部横向对比

对同一标段内同类项目，按专业和材料类型分组后计算：

- 综合单价变异系数 `CV = σ / μ`
- CV > 30% → 标注"同类项单价离散度大，疑似套项错误或特征差异未在单价中体现"
- 分组粒度建议：同一强度等级混凝土、同一规格钢筋、同一材质管材分别编组

### 8.2 含量指标反查

| 指标 | 计算方式 | 参考区间（常规民用建筑） | 偏离处理 |
|---|---|---|---|
| 钢筋含量 | 钢筋总量(kg) / 建筑面积(m²) | 框架 45-65、框剪 55-75、剪力墙 65-90、钢结构 35-55 | 偏离区间外 → 标注"含量偏离参考，需核实构件归并范围及建筑面积口径" |
| 混凝土含量 | 混凝土总量(m³) / 建筑面积(m²) | 框架 0.30-0.40、框剪 0.35-0.45、剪力墙 0.40-0.55 | 同上 |
| 模板含量 | 模板总量(m²) / 建筑面积(m²) | 2.5-4.0（按结构形式浮动） | 同上 |

> 注意：含量指标受结构形式、层数、设防烈度、地下室占比等因素影响显著，仅作第一道过滤，不直接得出"算量错误"结论。偏离时优先检查建筑面积口径是否一致（是否含地下室、是否按 GB/T 50353 计算）。

### 8.3 大金额项集中度检查

- TOP 20 项合价占比应 > 50% 且 < 85%
- 单项占比 > 20% → 标注"单项金额占比过高，重点核实工程量和单价"
- 单项占比 < 0.01% 且数量 > 100 项 → 标注"大量低价值散项，建议评估是否合并或归入其他项目"
- 占比计算口径：该项合价 / 分部分项工程费合计（不含措施、其他项目、规费、税金）

### 8.4 执行建议

- 上述规则在每次清单审核时自动执行，不需要用户单独要求
- 偏离标记均为"疑似"，不直接修改清单数据
- 发现异常后优先引导用户提供地区资料和图纸做进一步核实

## 九、输出要求

- 使用 `templates/boq-feature-check-table.md` 输出特征问题。
- 使用 `templates/boq-missing-items-register.md` 输出疑似漏项。
- 每次清单审核必须执行第八节的量价交叉验证，并在输出中附交叉验证结果摘要。
- 每条问题都要写依据来源：清单行号、设计说明页码、图号或招标范围条款。
- 不输出确定金额；如用户要求估算，必须单独列假设和依据。
