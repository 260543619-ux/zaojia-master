# 材料价差调整

## 一、调整原理

建筑材料费用一般占建安工程费的60-70%。市场价格波动直接影响造价。GB/T 50500-2024 明确：
- 合同约定的风险幅度内：承包人承担
- 超出风险幅度部分：发包人承担（或双方按比例承担）
- 禁止一刀切的"一次性包死"条款

## 二、三种调差方法

### 方法一：造价信息法（政府指导价）

**适用材料**：当地造价信息上定期发布价格的主要材料（钢筋、水泥、商品砼、砂石、电线电缆等）

**公式**：
```
涨价时: 材料价差 = 实际用量 × (施工期信息价 - 基准期信息价 × (1 + 风险幅度上限))
跌价时: 材料价差 = 实际用量 × (施工期信息价 - 基准期信息价 × (1 - 风险幅度下限))
```
*注：风险幅度通常为对称幅度（如±5%），即上限=下限。*

**流程**：
1. 确定风险幅度（一般±5%）
2. 价差超出风险幅度时触发调差
3. 仅对超出部分计取价差（承包人在风险幅度内的部分自担）

**示例**（涨价）：
- 钢筋基准期信息价：4000元/t
- 施工期信息价：4600元/t（上涨15%）
- 风险幅度：±5%（承包人自担200元/t）
- 可调价差 = 4600 - 4000×1.05 = 4600 - 4200 = 400元/t

**示例**（跌价）：
- 钢筋基准期信息价：4000元/t
- 施工期信息价：3600元/t（下跌10%）
- 风险幅度：±5%
- 可调价差 = 3600 - 4000×0.95 = 3600 - 3800 = -200元/t（发包人受益）

### 分月加权示例

适用于合同约定按施工期进场月份或使用月份加权平均的材料：

| 月份 | 用量(t) | 当月信息价(元/t) | 加权金额 |
|---|---:|---:|---:|
| 3月 | 40 | 4100 | 164000 |
| 4月 | 30 | 4300 | 129000 |
| 5月 | 30 | 4200 | 126000 |
| 合计 | 100 | — | 419000 |

加权施工期价 = 419000 / 100 = 4190 元/t。

若基准期价为 4000 元/t，合同约定承包人承担上涨 5% 内风险，则可调差单价：

```text
4190 - 4000 × (1 + 5%) = -10 元/t
```

结果为负或零时，说明未超过上涨风险幅度，不产生调增；是否扣减按合同调差条款判断。

### 方法二：市场询价法

**适用材料**：造价信息未覆盖的特殊材料、装饰材料、设备等

**步骤**：
1. 承包人提供至少3家供应商的同期报价单
2. 发包人/监理核实报价真实性
3. 双方协商确认调差单价
4. 按确认单价调整

**注意事项**：
- 报价单须为施工同期（不能事后补）
- 需考虑运距、运杂费、采保费等附加费用
- 特殊规格材料需提供生产厂家官方报价

### 方法三：综合系数法

**适用材料**：材料种类多、单项价格波动不大的辅助材料（砂、石、砖、商品砂浆等）

**公式**：
```
材料价差调整额 = (合同材料费 - 暂估价材料费) × 综合调差系数

综合调差系数 = ∑(各类材料权重 × 各类材料价格指数)
```

**特点**：
- 无需逐个材料计算，效率高
- 适用于品种繁杂的中小型项目
- 权重按投标报价中各类材料费占比确定

## 三、材料分类与风险幅度

| 材料类别 | 典型材料 | 建议风险幅度 | 调差方法 |
|---------|---------|------------|---------|
| 主要材料（Ⅰ类） | 钢筋、水泥、商品砼 | ±5% | 造价信息法 |
| 主要材料（Ⅱ类） | 电缆、管材、防水材料 | ±5% | 造价信息法/市场询价 |
| 装饰材料 | 石材、瓷砖、门窗 | ±5-10% | 市场询价法 |
| 安装设备 | 配电箱、水泵、电梯 | ±5% | 市场询价法 |
| 辅助材料 | 砂石、砖、灰浆 | ±10% | 综合系数法 |
| 周转材料 | 模板、脚手架 | 不调差（管理费含） | — |

## 四、调差程序

```
施工期材料采购
  │
  ├── 承包人在采购前7-14天申报材料品牌/规格/报价 → 发包人确认
  │
  ├── 确认实际用量（按合同计量规则）
  │     │
  │     └── 如投标时暂估数量 vs 实际数量，按实际数量调差
  │
  ├── 对比施工期价格 vs 基准期价格
  │     │
  │     ├── 价差 ≤ 风险幅度 → 不调整
  │     │
  │     └── 价差 > 风险幅度 → 计算调差金额
  │
  ├── 调差金额计入最近一期进度款
  │
  └── 竣工结算时汇总全部调差，扣除过程已支付的净额结算
```

## 五、基准期与施工期的界定

| 概念 | 定义 | 如何确定 |
|------|------|---------|
| 基准期 | 投标报价所依据的价格信息截止日期 | 招标文件中规定的造价信息期号（一般指投标截止日前28天的当期） |
| 施工期 | 材料实际采购/使用的时间段 | 以材料进场验收记录为准；主体结构材料按实际施工进度各月加权平均 |

**施工期价格确定方式（争议焦点）：**
- 按月调差：每月实际用量×当月信息价
- 按加权平均：总用量×∑(月用量×月价格)/∑月用量
- 按批次调差：每批材料进场时确认价格

*建议在合同中明确施工期价格的计算方式，避免结算争议。*

## 六、特例处理

### 6.1 价格波动极端情况

| 情况 | 处理方式 |
|------|---------|
| 单价涨跌幅超过30% | 双方协商可突破合同约定幅度，采用公平原则调整 |
| 某材料退出造价信息 | 改用市场询价法替代 |
| 政府限价/指导价变动 | 按政府文件调整 |
| 工期延误期间的涨价 | 责任方承担延误期间的涨价损失 |

### 6.2 甲供材价格风险

- 甲供材价格风险全部由发包人承担
- 承包人仅计取保管费（甲供材价值的1-2%）和安装费
- 甲供材损耗率按合同约定，超耗部分由承包人承担

### 6.3 暂估价材料

- 由发包人确认材料品牌、规格和单价
- 确认的材料单价与暂估价的差额，调整综合单价
- 承包人报价的利润和管理费不随暂估价调整而变化
- 发包人指定品牌但未指定价格的材料，以市场询价法确认

## 七、实务建议

1. **合同中明确调差范围**：列出可调差材料品类（表格式），注明每种材料的风险幅度和调差方法
2. **控制调差材料品种**：可调差材料控制在10-20种主要材料为宜，品种过多增加核算难度
3. **及时申报**：承包人应在施工期按合同约定的调差周期（按月/按季）申报，而非竣工时一次性申报
4. **保留凭证**：材料采购合同、发票、进场验收单、造价信息截图等作为调差依据
5. **避免双口径调差**：材料费已在综合单价中包含了基础价格，调差只调超出风险幅度部分的增量
