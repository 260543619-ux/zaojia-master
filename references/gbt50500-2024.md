# GB/T 50500-2024 《建设工程工程量清单计价标准》核心解读

## 标准基本信息

| 项目 | 内容 |
|------|------|
| 标准编号 | GB/T 50500-2024 |
| 标准名称 | 建设工程工程量清单计价标准 |
| 发布部门 | 住房和城乡建设部 |
| 发布日期 | 2024年11月26日 |
| 实施日期 | 2025年9月1日 |
| 替代标准 | GB 50500-2013（同步废止） |
| 标准性质 | 推荐性国家标准（GB/T） |
| 主编单位 | 住房和城乡建设部标准定额研究所 |

## 四大根本性变化

### 1. 名称变更
- 旧："建设工程工程量清单计价**规范**"（Code）
- 新："建设工程工程量清单计价**标准**"（Standard）

### 2. 效力降级
- 旧：强制性国标 GB → 新：推荐性国标 GB/T
- 不再具有普遍强制约束力，但国有资金投资工程仍应主要适用
- 发承包双方通过合同约定引用，仍可产生约束力

### 3. 计价模式市场化
- 取消定额编制：最高投标限价、投标报价不再将政府定额和信息价作为主要计价依据
- 改用：市场价格信息 + 工程造价数据库 + 投标人自身装备/管理/成本数据
- 核心导向："让价格主要由市场说了算"

### 4. 除税价模式
- 综合单价明确为**不含增值税的税前全费用价格**
- 增值税在单位工程汇总表中单独计列

## 各章节核心条款解读

### 第1章 总则

- **1.0.1** 目的：完善市场化工程造价计价体系
- **1.0.2** 适用范围：新建、改建、扩建工程
- **1.0.3** 国有资金占控股或主导地位的建设工程，应执行本标准
- **1.0.4** 非国有资金项目可参照执行

### 第2章 术语

新增关键术语（共30个术语，其中以下为新增或重要修改）：

| 术语编号 | 术语 | 说明 |
|---------|------|------|
| 2.0.4 | 安全生产措施费 | 原"安全文明施工费"，口径收紧 |
| 2.0.10 | 工程量清单缺陷 | 清单中存在的错项、漏项、特征描述不符等 |
| 2.0.12 | 费率计价 | 以计费基数乘以费率确定价格 |
| 2.0.18 | 合同清单 | 已标价工程量清单，合同文件组成部分 |
| 2.0.21 | 施工过程结算 | 施工过程中对已完工程进行计量、计价和支付 |
| 2.0.22 | 合同图纸 | 作为合同组成部分的图纸 |
| 2.0.23 | 合同规范 | 作为合同组成部分的技术标准与规范 |
| 2.0.24 | 施工深化设计 | 承包人对图纸的深化和细化 |
| 2.0.30 | 新增工程 | 合同范围外的新增工程，区别于工程变更 |

### 第3章 工程量清单编制

**3.1 一般规定**
- 工程量清单由招标人（或其委托的造价咨询人）负责编制
- 清单应按单位工程、专业工程分别编制
- 项目编码采用12位阿拉伯数字：01-02-03-XXX-XXX（专业-分部-分项-顺序码）

**3.2 分部分项工程项目清单（核心变化）**
- 必须载明：项目编码、项目名称、项目特征描述、计量单位、工程量
- **项目特征描述**是区分不同清单项目的关键，描述不清是纠纷的首要原因
- 工程量计算规则按各专业附录执行

**3.3 措施项目清单**
- 分为：安全生产措施费（原安全文明施工费，不可竞争）和其他措施项目（可竞争）
- 能计量的措施项目（如脚手架）应列工程量；不能计量的（如环境保护）按项列

**3.4 其他项目清单（"5改4"变化）**
- 暂列金额（由招标人根据工程复杂程度估算）
- 暂估价（含材料暂估价、设备暂估价、专业工程暂估价）
- 计日工（招标人列出人工、材料、机械的暂定数量）
- 总承包服务费（对发包人直接发包专业工程的管理/配合/协调服务费）
- ~~规费~~：已并入人工费/管理费
- ~~税金~~：已改为增值税单独计列

### 第4章 最高投标限价（原"招标控制价"）

- 国有资金投资的工程，应编制最高投标限价
- 最高投标限价应在招标文件中公布，不得上浮
- 以市场价格信息、工程造价数据库为主要依据编制
- 不再强制依赖政府定额

### 第5章 投标报价

- 投标人自主确定投标报价，但不得低于成本
- 安全生产措施费（原安全文明施工费）、税金不得作为竞争性费用
- *注：GB/T 50500-2024将原2013版"规费"并入人工费/管理费，不再作为独立竞争性费用列项*
- 投标报价不得高于最高投标限价，否则应否决其投标

### 第6章 清标（新增章节）

**核心流程：**
1. 评标委员会在推荐中标候选人前，对投标报价进行澄清或说明
2. 审查范围：算术误差、细微偏差、报价合理性、报价完整性（漏报/未报）
3. 投标人应在规定时间内书面回复
4. 澄清不得改变实质性内容（工程范围、工期、合同条件、投标总价）
5. 澄清文件可作为合同文件组成部分

**清标重点审查项：**
- 不平衡报价（偏离评标基准价15-20%以上）
- 算术误差（单价×工程量≠合价）
- 漏项或多项
- 综合单价明显偏离市场水平
- 不可竞争费用被下浮

### 第7章 合同价款的约定

- 合同类型应在招标文件中明确
- 单价合同：发包人对工程量清单的完整性和准确性负责
- 总价合同：投标人对工程量清单的完整性和准确性负责
- 措施项目清单的完整性、准确性，不论何种合同均由承包人承担

### 第8章 合同价款的调整

主要调整情形：
- 工程量清单缺陷（错项、漏项）：按规定调整
- 工程变更：按合同约定的定价原则
- 物价变化：超过约定幅度时调整（详见附录A）
- 暂估价确认：按合同约定确认后调整
- 计日工：按实际签证确认
- 新增工程：按合同约定或双方协商定价
- **禁止"无限风险"条款**：不得采用"所有风险由承包人承担"等表述

### 第9章 施工过程结算（新增章节）

- 节点划分应在合同中明确（如：按月、按楼层、按形象进度）
- 发包人应在收到过程结算文件后约定时间内完成审核
- 过程结算文件作为竣工结算的依据
- 竣工结算时仅对过程结算未覆盖部分进行补充

### 第10章 竣工结算

- 承包人应在工程竣工验收合格后约定时间内提交竣工结算文件
- 发包人审核时限：500万以下20天，500-2000万30天，2000-5000万45天，5000万以上60天
- 逾期未回复视为认可

### 第11章 争议解决

- 新增：**争议评审制度**——双方委托争议评审委员会（或机构）进行评审
- 争议评审意见可作为调解或裁决的参考
- 保留：协商、调解、仲裁、诉讼等传统途径
- 删除：原"监理或造价工程师暂定"条款

### 第12章 竣工结算与支付

- 发包人应在收到竣工结算文件后约定时间内支付结算款
- 质量保证金不得超过工程价款结算总额的3%
- 缺陷责任期满后，发包人应在14天内退还剩余质量保证金

## 附录体系

- **附录A**：物价变化合同价款调整方法（价格指数调差法+价格信息调差法）
- **附录B**：工程量清单缺陷调整方法
- **附录C**：工程变更计价方法
- **附录D～G**：各专业工程量计算规则（房建/装饰/安装/市政/园林）

## 2013版 vs 2024版关键差异对比表

| 对比维度 | GB 50500-2013 | GB/T 50500-2024 |
|---------|--------------|----------------|
| 标准性质 | 强制性 GB | 推荐性 GB/T |
| 标题用词 | 规范 | 标准 |
| 计价基础 | 政府定额+信息价为主 | 市场价格+企业定额为主 |
| 工程量清单组成 | 5部分（含规费税金） | 4部分（规费并入人/管费，税金单独） |
| 综合单价内含 | 含税金 | 不含增值税 |
| 安全文明施工费名称 | 安全文明施工费 | 安全生产措施费 |
| 投标限价名称 | 招标控制价 | 最高投标限价 |
| 清标制度 | 无专门规定 | 专章规定 |
| 施工过程结算 | 无 | 专章规定 |
| 争议解决 | 监理暂定+调解+仲裁 | 争议评审+调解+仲裁 |
| 新增工程 vs 变更 | 统称变更 | 严格区分 |
| 工程签证 | 有 | 删除，以更严谨规则替代 |
| BIM应用 | 无 | 增加条款 |
| 无限风险条款 | 无明确禁止 | 明确禁止 |
| 甲供材 | 进入综合单价后扣除 | 不进入综合单价 |

## 使用本标准的实务提示

1. **过渡期（2025年9月1日前）**：仍适用 GB 50500-2013；过渡期内签订的合同应明确适用的标准版本
2. **新旧标准合同衔接**：旧标准下签订的合同，原则上仍按旧标准执行完毕，除非双方协商变更
3. **清标制度落地**：招标文件中应明确清标程序和处理规则
4. **过程结算条款**：合同中必须约定过程结算节点，否则承包人可按合理节点申请
5. **增值税处理**：编制最高投标限价和投标报价时，务必区分含税价和不含税价，防止实务混乱
