# 国内主流造价软件与定额体系

## 一、广联达（Glodon）系列

广联达是国内工程造价领域市场占有率最高的软件平台（约70%+），核心产品线覆盖计量与计价。

### 核心产品

| 软件 | 全称 | 功能 | 适用专业 |
|------|------|------|---------|
| GTJ | 广联达土建算量 | 工程量计算 + 钢筋翻样 | 房屋建筑、装饰、钢结构 |
| GQI | 广联达安装算量 | 安装工程量计算 | 电气、给排水、暖通、消防 |
| GCCP | 广联达云计价 | 清单计价 + 定额计价 | 土建、安装、市政、园林、轨道等 |
| GBQ | 广联达计价（旧版） | 计价（逐步被GCCP替代） | 同上 |
| GMA | 广联达精装算量 | 装饰装修工程量 | 精装修 |

### GTJ 操作要点

- 导入CAD图纸（dwg格式）自动识别构件
- 支持导入Revit模型（RVT格式）进行BIM算量
- 建模顺序：轴网 → 柱 → 梁 → 板 → 墙 → 基础 → 装饰
- 汇总计算后可导出工程量清单和钢筋明细表
- 计算规则可在软件中选择"清单规则"或"定额规则"

### GCCP 操作要点

- 新建项目 → 选择定额库（按省份选择）→ 选择专业（土建/安装/市政等）
- 导入Excel格式的工程量清单（广联达格式，非通用Excel）
- 套定额：在定额库中搜索定额编号或关键词
- 调价：可手动调整人工单价、材料单价，也可导入当地信息价文件（.xxj格式）
- 导出：可导出综合单价分析表、工程量清单计价表、规费税金表（旧版软件用词）等

### AI 对接导出建议

从 GTJ/GCCP 等软件导出给 AI 检查时，建议同时提供以下表格或截图：

| 资料 | 用途 |
|---|---|
| 项目设置截图 | 确认地区、定额库、费用文件、税率、计价模板 |
| 分部分项清单计价表 | 检查清单五要素、工程量、综合单价、合价 |
| 综合单价分析表 | 检查人工、材料、机械、管理费、利润、风险费 |
| 定额子目明细表 | 区分清单工程量和定额工程量，检查套项和换算 |
| 人材机汇总表 | 检查材料价格来源和主材异常 |
| 措施/其他项目/税金表 | 检查不可竞争费、暂列暂估、增值税口径 |

常见风险：

- 清单工程量、定额工程量、主材工程量字段混用。
- 软件模板保留“规费税金”等旧称，但实际费用内容已按新口径处理。
- 导出 Excel 删除了隐藏列、公式或项目设置，导致 AI 无法复核来源。

## 二、鲁班（Luban）系列

鲁班是国内较早的BIM+造价一体化平台，长三角地区市场份额较高。

| 软件 | 功能 |
|------|------|
| 鲁班土建 | 土建工程量计算 |
| 鲁班钢筋 | 钢筋翻样与计算 |
| 鲁班安装 | 安装工程量计算 |
| 鲁班造价 | 计价软件 |
| Luban BIM | BIM平台（碰撞检测、进度模拟等） |

**鲁班 vs 广联达差异点：**
- 鲁班更侧重BIM全生命周期应用
- 广联达在计价端（GCCP）市场份额更高
- 鲁班在上海/浙江/江苏有地域优势

## 三、斯维尔（Thsware）系列

深圳斯维尔科技——南方区域（广东/深圳/广西）市场占有率较高。

| 软件 | 功能 |
|------|------|
| 斯维尔三维算量 | 土建+钢筋一体化算量 |
| 斯维尔安装算量 | 安装算量 |
| 斯维尔清单计价 | 计价（支持广东省定额和深圳定额） |

**特点**：与深圳市建设工程造价管理站的造价信息系统深度对接。

## 五、定额体系

### 5.1 定额层级

```
全国统一基础定额（住建部发布，基础标准）
  ├── 省级建设工程计价定额（各省住建厅发布，实际执行依据）
  │     ├── 建筑与装饰工程消耗量定额
  │     ├── 安装工程消耗量定额
  │     ├── 市政工程消耗量定额
  │     ├── 园林绿化工程消耗量定额
  │     └── 城市轨道交通工程概算定额
  └── 企业定额（大型施工企业内部使用，市场化方向）
```

### 5.2 主要省份定额体系

| 省份 | 定额名称/版本 | 特点 |
|------|------------|------|
| 北京 | 北京市建设工程计价依据（2021版） | 人工+机械为管理费取费基数 |
| 上海 | 上海市建设工程预算定额（2016版） | 能源消耗量定额单列 |
| 广东 | 广东省建设工程计价依据（2018版） | 人工费为管理费取费基数；动态人工费调整系数 |
| 深圳 | 深圳市建设工程计价标准 | 独立于广东省定额 |
| 浙江 | 浙江省建设工程计价依据（2018版） | 人工价格指数制度 |
| 江苏 | 江苏省建设工程费用定额（2014版） | 安全生产措施费分市区/县镇两档 |
| 四川 | 四川省建设工程工程量清单计价定额（2020版） | 明确海拔高度调整系数 |
| 重庆 | 重庆市建设工程计价定额（2018版） | 山地建筑增加费有专门规定 |

### 5.3 定额子目结构解读

每个定额子目包含：

```
定额编号: 1-1-1（专业-分部-子目）
项目名称: 人工挖土方
工作内容: 包括挖土、抛土、修底、修边
单位: 100m³
消耗量:
  ├── 人工: 综合工日 X个工日
  ├── 材料: 无
  └── 机械: 无
基价: 人工工日×人工单价 + ∑(材料量×材料单价) + ∑(机械台班×台班单价)
```

### 5.4 定额计价到市场价的转换

GB/T 50500-2024 推行市场化后，从定额价到市场价的转换路径：

1. **直接采用市场价**（钢筋、商品砼等有活跃市场的材料）—— 首推
2. **定额基价 × 调整系数**（根据造价信息发布的调价系数）
3. **定额消耗量 × 市场单价**（定额消耗量保留，替换单价来源）
4. **企业实际成本数据替换**（大型企业采用企业定额）

## 六、造价信息网

各省市住房和城乡建设主管部门定期发布：

| 信息类型 | 发布周期 | 内容 |
|---------|---------|------|
| 人工费指导价 | 季度/半年 | 建筑/安装/市政等各工种指导价 |
| 材料信息价 | 月度/双月 | 主要建材的市场平均价（含税/不含税） |
| 机械台班参考价 | 半年/年 | 主要施工机械台班费用 |
| 造价指标 | 季度/年 | 各类建筑的造价指标（元/㎡） |
| 价格指数 | 季度/半年 | 建安工程价格指数（环比/同比） |

## 七、实务操作建议

1. **新项目开工**：首先确认项目所在地，并提示用户上传对应省/市定额库版本、费用文件和软件导出的项目设置截图
2. **定额选择**：如工程所在地为深圳，应优先用深圳定额而非广东省定额；如无本地定额，可用省级定额，并在输出中说明依据来源
3. **信息价订阅**：及时获取最新的当地造价信息；未上传信息价文件时，只能输出需补资料清单和价格校核思路
4. **软件数据互通**：广联达GTJ算量 → 导出Excel → 导入GCCP计价是最常见的流程
5. **定额争议**：如定额中无适用子目，可借用类似子目并申请定额解释，或采用市场询价+补充定额方式处理
