# 信息价缺失时的市场询价方法

本文件用于当项目所在地工程造价信息（信息价）未取得时，如何进行合理的市场价格暂估和替代验证。它不替代正式投标报价，而是帮助形成可追溯的估算工作底稿。

## 一、定价数据源优先级

| 优先级 | 数据源 | 适用于 | 可信度 |
|---|---|---|---|
| 1 | 项目所在地当期工程造价信息（信息价） | 钢材、商品砼、水泥、砂石、电缆等 | 高 |
| 2 | 专业厂家/供应商正式报价单（≥3 家） | 配电箱柜、水泵、风机、消防设备、门窗 | 高 |
| 3 | 企业历史项目采购合同/结算数据 | 类似项目、类似规格 | 中高 |
| 4 | 省级造价信息网近期发布价 + 地区系数调整 | 信息价未覆盖的辅助材料 | 中 |
| 5 | 全国参考指标 + 地区系数 + 时间修正 | 估算和快速参考模式 | 低（仅用于摸底） |

## 二、信息价缺失时的暂估方法

### 当用户未上传当期信息价时

按以下流程降级处理：

```
1. 询问用户：是否有成都市工程造价信息（X月期）？
   ├── 有 → 采用信息价，标注期号
   └── 无 → 继续

2. 询问用户：是否有厂家/供应商报价？
   ├── 有 ≥3 家 → 取中位价，标注来源
   └── 无 → 继续

3. 使用全国参考指标 × 地区系数调整
   - 必须标注"暂估价"而非"审定单价"
   - 必须输出"需补资料清单"含"当期信息价"
   - 必须标注价格有效期（建议30天内更新）
```

### 地区系数参考

| 城市等级 | 调整系数 | 说明 |
|---|---|---|
| 一线（北上广深） | 1.15 - 1.30 | 人工费、管理费偏高 |
| 新一线（成都、杭州、武汉等） | 1.00 | 基准 |
| 二线（省会城市） | 0.90 - 1.00 | |
| 三四线 | 0.80 - 0.90 | |

### 时间修正

若参考指标数据年份与当前时间差距 > 12 个月，应在调价说明中注明"需更新地区价格资料"，并建议用户提供最新一期的当地造价信息。本 Skill 内置参考指标的时效性标注见 `references/zaojia-indicators.md` 的更新提醒。

## 三、主要材料暂估价参考方法

### 钢材

按全国均价 + 成都地区偏差 ≈ 全国均价 × 1.02（成都钢材略高于全国均价）。以 2025 年 5 月全国均价为基准，螺纹钢 HRB400 Φ16-25 参考区间 3,750-4,100 元/t。

### 商品混凝土

成都地区 C30 参考 450-550 元/m³，C35 参考 480-580 元/m³。**注意**：商品砼价格含泵送和不含泵送差异约 15-25 元/m³，需确认合同清单中泵送费的计价方式。

### 电缆

铜芯电缆对铜价（上海有色网 SMM 1# 电解铜）高度敏感。暂估时应注明铜价基准日。小截面（≤6mm²）和大截面（≥70mm²）的每米单价跨几个数量级，必须按具体规格分别估价。

### 装配式预制构件

预制柱/叠合板/楼梯单价含模具摊销、运输、吊装和灌浆。成都地区装配式预制叠合楼板 C35 暂估参考 2,200-2,800 元/m³（含运输吊装），不含深化设计费。预制柱 C40 暂估参考 2,500-3,200 元/m³。

## 四、定价输出格式要求

即使是在暂估状态下，每项报价也必须标注：

| 字段 | 要求 |
|---|---|
| 单价 | 数字 + 单位 |
| 价格来源 | "成都市信息价2026年X期" / "XX供应商报价2026-X-X" / "全国参考区间×成都系数暂估" |
| 是否含税 | 明确标注"不含税综合单价"（对齐 GB/T 50500-2024） |
| 有效期建议 | "建议30天内以当期信息价或正式报价替换" |
| 风险提示 | 如价格波动敏感（电缆/钢材），注明"需关注近期铜价/钢价走势" |

## 五、诚实表述模板

### 当全部采用暂估价时

在报告"价格资料参考"或"编制说明"章节注明：

> "本次报价中，主要材料价格未取得当期《XX市工程造价信息》正式价格表。钢筋、混凝土、电缆、设备等价格按全国参考区间 × 地区系数暂估。
>
> **正式投标或控制价编制前，必须以以下资料替换：**
> 1. XX市工程造价信息 XXXX 年 X 期
> 2. 厂家正式报价单（≥3 家）
> 3. 四川 2020 定额及当期取费文件
> 4. XX 计价软件项目设置和费用模板"

### 当部分材料有信息价、部分暂估时

在报告中逐项标注来源列：

| 材料 | 单价 | 来源 | 可信度 |
|---|---|---|---|
| 螺纹钢 HRB400 Φ16-25 | 4,150 元/t | 成都市信息价 2026年4月期 | 高 |
| 配电箱 AP1 | 38,000 元/台 | 3家供应商报价中位数 | 中 |
| 电缆 YJV-4×25+1×16 | 95 元/m | 全国参考区间 × 成都系数暂估 | 低 — 需信息价替换 |
