# 钢筋平法与造价影响基础

本文件用于钢筋工程量异常识别和算量指标反查，不替代 16G101、22G101 等正式图集、结构设计说明、算量软件或专业钢筋翻样复核。

## 一、对造价影响最大的平法要素

| 要素 | 影响内容 | AI 检查点 |
|---|---|---|
| 保护层厚度 | 影响钢筋下料长度、箍筋尺寸 | 是否按构件类型和环境类别设置 |
| 锚固长度 | 影响梁、板、柱、墙端部钢筋长度 | 是否按钢筋等级、直径、抗震等级取值 |
| 搭接长度 | 影响搭接区钢筋工程量 | 是否区分绑扎搭接、机械连接、焊接 |
| 接头形式 | 影响接头数量和单价 | 机械连接是否单列或计入钢筋综合单价 |
| 抗震等级 | 影响锚固、搭接、箍筋加密 | 结构说明与软件设置是否一致 |
| 箍筋加密区 | 影响箍筋数量 | 梁端、柱端、节点区是否按规则识别 |
| 措施筋 | 马凳筋、定位筋、拉结筋等 | 是否按合同/清单/定额口径计取 |
| 损耗率 | 影响定额消耗或主材用量 | 是否重复计入软件量和定额损耗 |

## 二、常见异常信号

| 异常 | 可能原因 | 需复核资料 |
|---|---|---|
| 钢筋 kg/m2 明显偏高 | 地下室、人防、转换层、抗震等级高、措施筋重复计入 | 结构说明、楼层分区指标、GTJ 设置 |
| 钢筋 kg/m2 明显偏低 | 构件漏建、板筋/墙筋漏识别、接头未计 | GTJ 构件汇总、钢筋明细表 |
| 同类楼层钢筋含量差异大 | 构件识别错误、标高/楼层复制错误 | 分楼层钢筋汇总 |
| 梁柱节点钢筋异常 | 锚固、加密区、节点构造设置异常 | 平法图集版本、结构总说明 |

## 三、AI 输出边界

- 可以提示“需复核平法设置、抗震等级、搭接/锚固、措施筋口径”。
- 可以用含钢量指标做异常筛查。
- 不得直接替代钢筋翻样或图集精算。
- 不得凭经验修改钢筋工程量；必须回到结构图、图集、软件设置和人工复核。

## 四、建议用户上传资料

- 结构设计总说明、构件配筋图、平法图集版本说明。
- GTJ 钢筋工程量汇总表、分楼层钢筋明细表、软件计算设置截图。
- 合同或清单中关于钢筋损耗、接头、措施筋的计价口径。
