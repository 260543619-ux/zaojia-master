# 竣工结算、竣工决算与造价审计

## 一、竣工结算

### 1.1 结算文件组成

承包人提交的竣工结算文件应包含：
1. 竣工结算书封面和签署页
2. 竣工结算总说明
3. 建设项目竣工结算汇总表
4. 单项工程竣工结算汇总表
5. 单位工程竣工结算汇总表
6. 分部分项工程量清单计价表（结算版）
7. 措施项目清单计价表（结算版）
8. 其他项目清单计价表（结算版）
9. 增值税计价表
10. 综合单价调整表（标注变化原因）
11. 变更签证汇总表（按编号排列，附变更令和签证单）
12. 材料价差调整计算表
13. 索赔费用汇总表
14. 工期奖惩明细
15. 甲供材对账表
16. 已支付进度款汇总及凭证
17. 质量保证金计算

### 1.2 结算审核流程

```
承包人提交竣工结算文件
  │
  ├── 发包人签收并记录提交日期
  │
  ├── 发包人/监理/造价咨询进行审核
  │     │
  │     ├── 审核时限：
  │     │   500万以下 → 20天
  │     │   500-2000万 → 30天
  │     │   2000-5000万 → 45天
  │     │   5000万以上 → 60天
  │     │
  │     ├── 发现问题的，发函要求承包人补充资料或说明
  │     │
  │     └── 逾期未回复 → 视为认可承包人结算文件
  │
  ├── 双方对审核结果达成一致 → 签署结算定案表
  │
  ├── 发包人支付结算款（扣除质保金和已付进度款）
  │
  └── 发包人逾期未支付 → 按同期贷款市场报价利率(LPR)支付利息
```

### 1.3 常见审减点

| 审减类型 | 具体表现 | 审减率参考 |
|---------|---------|----------|
| 工程量多算 | 重算、多算、错算 | 5-15% |
| 综合单价高套 | 套用高价子目，实际施工标准低于套用标准 | 3-8% |
| 取费标准偏高 | 管理费/利润超过合同约定比例 | 2-5% |
| 变更签证虚报 | 无指令变更、签证数量虚增、事后补签 | 3-10% |
| 材差调整不合理 | 用量超实际、基价错误、未应调但调了 | 2-5% |
| 索赔证据不足 | 口头指令、失去时效、缺少关键证据 | — |
| 甲供材未扣回 | 未在结算中扣除已供材料费 | — |
| 规费税金 | 不可竞争费被下浮后还原 | 1-3% |
| 总体审减率 | — | 10-20%（正常范围） |

### 1.4 结算争议高频点

1. **隐蔽工程**：已覆盖不可复核，需依靠过程签证和影像资料
2. **变更签证有效性**：签字人权限不足、口头指令、超时效
3. **综合单价适用争议**：是否属于"类似"价格的判定标准
4. **材料价差**：施工期价格如何确定（按月？按加权平均？）
5. **停窝工损失**：人工和机械闲置费的计算标准
6. **措施费包干**：是否可因工程量变化调整措施费

## 二、竣工决算

### 2.1 结算 vs 决算

| 维度 | 竣工结算 | 竣工决算 |
|------|---------|---------|
| 范围 | 仅建安工程费 | 全部建设投资 |
| 编制主体 | 承包人编制 | 建设单位编制 |
| 文件内容 | 结算书+支持文件 | 财务报表+说明书+审计报告 |
| 目的 | 确定应付工程款 | 确定固定资产入账价值 |
| 包含费用 | 建安费 | 建安费+设备购置费+工程建设其他费+预备费+建设期利息+铺底流动资金 |

### 2.2 决算文件组成

1. 竣工财务决算报表
   - 建设项目竣工财务决算审批表
   - 竣工财务决算总表
   - 交付使用资产总表
   - 交付使用资产明细表
2. 竣工财务决算说明书
3. 竣工结算对比表
   - 结算价 vs 批复概算
   - 超概算的要分析原因

### 2.3 工程建设其他费（决算的重要组成部分）

| 费用类别 | 说明 |
|---------|------|
| 土地征用及拆迁补偿费 | 按实际发生金额 |
| 建设单位管理费 | 按总投资一定比例 |
| 勘察设计费 | 按合同实际结算 |
| 监理费 | 按合同实际结算 |
| 造价咨询费 | 按合同实际结算 |
| 研究试验费 | 按合同实际结算 |
| 联合试运转费 | 按实际消耗 |
| 工程保险费 | 按合同实际结算 |
| 招标代理服务费 | 按合同实际结算 |

## 三、造价审计

### 3.1 审计类型

| 审计类型 | 实施主体 | 法律效力 |
|---------|---------|---------|
| 政府审计 | 审计机关（审计署/审计厅局） | 对国有资金项目有强制约束力 |
| 内部审计 | 建设单位内审部门 | 企业管理层面约束力 |
| 第三方审计 | 造价咨询机构 | 依据委托合同产生约束力 |

### 3.2 审计重点内容

#### 工程量审计
- 核对竣工图与现场实际情况
- 核对工程量计算书与计量规则
- 抽查重点子项的工程量（重点抽查造价占比前20%的子项）

#### 综合单价审计
- 单价是否与合同（中标价）一致
- 变更新增单价是否按规定组价
- 管理费率/利润率是否超过投标时水平

#### 变更签证审计
- 变更是否有发包人书面指令
- 签证内容是否在合同范围内已包含
- 签证数据是否真实（结合现场照片、监理日志交叉验证）
- 签证签字人是否有权限

#### 材料价差审计
- 调差材料是否在合同约定的可调差范围内
- 实际用量是否与材料台账、进场验收记录相符
- 施工期价格认定方式是否符合合同约定

#### 索赔审计
- 索赔事件是否真实发生
- 索赔时效是否满足合同约定
- 工期影响分析是否合理（关键线路上的延误才能索赔工期）
- 费用计算是否依据充分

### 3.3 审计常见问题

1. **"审减率至上"误区**：过度追求审减率可能导致不合理的低结算
2. **资料不全审计**：缺乏关键证据时的推定应当有利于施工人
3. **定额思维审计**：GB/T 50500-2024推行市场定价后，审计不宜以定额为唯一标准
4. **争议久拖不决**：应及时走争议评审或调解程序，不应无限期搁置

## 四、争议解决

### 4.1 争议解决路径（GB/T 50500-2024）

```
争议发生
  │
  ├── 1. 协商（双方自行沟通）
  │
  ├── 2. 争议评审（委托争议评审委员会/机构，新标准新增）
  │     └── 评审意见出具后，双方可接受或异议
  │
  ├── 3. 调解（第三方调解机构）
  │
  ├── 4. 仲裁（合同约定仲裁机构，一裁终局）
  │
  └── 5. 诉讼（合同未约定仲裁的，向管辖法院提起诉讼）
```

### 4.2 争议评审制度（新增）

- 双方在合同中约定评审委员会（或机构）
- 评审委员由双方共同委托或各自指定+共同选定首席
- 评审意见不具有强制约束力，但可作为后续仲裁/诉讼的证据
- 与FIDIC 2017版的DAAB（争议避免/裁决委员会）理念接近
