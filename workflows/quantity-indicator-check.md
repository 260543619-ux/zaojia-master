# 算量指标反查 Agent 工作流

## 适用用户

造价员、预算员、成本管理人员、项目商务。

## 输入文件

- GTJ/GCCP 或其他算量/计价软件导出的工程量表
- 建筑面积、地上/地下指标、结构类型、层数、檐高
- 钢筋、混凝土、模板、砌体等主要工程量
- `references/zaojia-indicators.md` 中的含量指标作为辅助校核

## 执行步骤

1. 抽取建筑面积、结构类型、地下层数和主要工程量。
2. 计算关键含量指标：钢筋 kg/m2、混凝土 m3/m2、模板 m2/m2、砌体 m3/m2。
3. 与项目类型相近的参考区间进行对比。
4. 标记明显偏高/偏低的工程量，并要求用户回查模型设置、计算规则或构件范围。
5. 输出算量合理性检查表和需复核构件/楼层/专业。

## 中间判断

- 地下室、人防、超高层、转换层、钢结构、装配式等是否导致指标偏离。
- 软件是否混用了清单工程量和定额工程量。
- 统计面积是否与指标计算口径一致。
- 钢筋平法规则、搭接、锚固、措施筋和损耗设置是否影响结果。

## 输出成果

- `templates/quantity-indicator-check-table.md`
- `templates/need-more-documents-list.md`
- `templates/management-summary.md`

## 风险控制

- 指标反查只用于发现异常，不直接判定算量错误。
- 不替代 GTJ、BIM、手算或专业算量复核。
- 指标区间必须结合结构形式、地下层数和设计标准解释。

## 人工确认点

- 建筑面积口径。
- 模型范围和构件识别设置。
- 平法规则版本、钢筋接头和措施筋设置。
