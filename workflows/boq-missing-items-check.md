# 清单特征与漏项检查 Agent 工作流

## 适用用户

造价员、预算员、招投标商务人员、清单编制/审核人员。

## 输入文件

- 工程量清单 Excel
- 招标范围、图纸目录、设计说明、材料做法表
- 专业工程界面划分、暂估价和专业分包范围
- 已标价清单或类似项目清单（可选）

## 执行步骤

1. 识别项目专业：土建、装饰、安装、市政、园林或其他。
2. 检查清单五要素：编码、名称、项目特征、单位、工程量。
3. 按 `references/boq-check-rules.md` 检查特征描述完整性。
4. 对照招标范围、图纸说明和常见分部分项，列出疑似漏项。
5. 检查单位、精度、重复编码、措施/其他项目口径。
6. 输出清单特征检查表、漏项登记表和需确认问题。

## 中间判断

- 图纸或说明出现的材料/构造是否在清单特征中体现。
- 影响综合单价的规格、强度、厚度、品牌、部位是否缺失。
- 疑似漏项是否可能已由专业分包、暂估价、总价措施或其他清单覆盖。
- 清单缺陷是否会影响报价、结算或变更争议。

## 输出成果

- `templates/boq-feature-check-table.md`
- `templates/boq-missing-items-register.md`
- `templates/need-more-documents-list.md`

## 风险控制

- 没有完整图纸和招标范围时，只能输出“疑似漏项”。
- 不直接给漏项金额；如需估算，必须说明依据和假设。
- 不凭空补定额编号或项目编码。

## 人工确认点

- 专业界面、甲供材、暂估价、独立分包范围。
- 是否属于措施项目、合同包干内容或发包人另行委托内容。
- 项目特征是否允许引用图纸详见而不逐项展开。
