# 变更签证审核 Agent 工作流

## 适用用户

项目商务、结算员、造价咨询审核人员、成本管理人员。

## 输入文件

- 变更单、签证单、联系单、设计变更通知
- 合同变更签证条款、授权签字人约定
- 对应清单项、图纸、计算书、照片、会议纪要、监理日志

## 执行步骤

1. 识别事项类型：工程变更、新增工程、现场签证、索赔或反索赔。
2. 抽取编号、日期、事项、部位、数量、单价、金额、签署和附件。
3. 检查四要素：及时性、准确性、完整性、授权性。
4. 核对是否合同范围内已包含、是否重复计取、是否进入进度款或结算。
5. 检查单价适用顺序：已有价、类似价、新组价、市场询价。
6. 输出审核表、待补资料和人工确认点。

## 中间判断

- 是否有发包人或合同约定主体的书面指令。
- 是否有工程量计算依据和附件证据。
- 签证内容是否与原清单、措施费、其他项目重复。
- 时效和签字权限是否需结合合同专用条款确认。

## 输出成果

- `templates/change-visa-review-table.md`
- `templates/need-more-documents-list.md`
- `templates/settlement-deduction-register.md`

## 风险控制

- 缺少合同条款时，不直接认定过期、无效或丧失权利。
- 现场照片只能作为辅助证据，不能替代工程量计算。
- 争议项使用“建议暂列争议金额”而非最终审减。

## 人工确认点

- 现场实际是否发生。
- 签字人是否具备合同授权。
- 计量口径、类似价适用性和争议处理策略。
