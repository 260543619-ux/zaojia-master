# 进度款审核 Agent 工作流

## 适用用户

项目商务、预算员、成本管理人员、监理/咨询审核人员、项目经理。

## 输入文件

- 合同付款条款、进度款申报表、已完工程量报表
- 形象进度确认单、监理确认记录、工程量计算书
- 已标价工程量清单、变更签证台账、材料调差申报
- 预付款支付和扣回表、质保金扣留表、甲供材扣回表

## 执行步骤

1. 抽取本期申报金额、累计申报金额、合同价、付款比例和审核周期。
2. 核对本期已完工程量与形象进度、监理确认和清单工程量。
3. 检查本期变更签证是否已确认，未确认事项不得作为确定付款依据。
4. 按合同公式计算本期应付：已完产值 x 支付比例 + 已确认变更/调差 - 预付款扣回 - 甲供材扣回 - 质保金 - 其他扣款。
5. 检查累计付款是否超过合同约定比例或已完产值。
6. 输出进度款审核表、超报/少报问题、需补资料和人工确认点。

## 中间判断

- 已完工程量是否超过合同清单量或现场确认量。
- 形象进度与产值比例是否明显不匹配。
- 预付款扣回节点和扣回比例是否正确。
- 质保金是否按合同基数和比例扣留。
- 甲供材是否按本期耗用或合同约定扣回。
- 本期变更签证是否存在未确认先支付。

## 输出成果

- `templates/progress-payment-audit-table.md`
- `templates/need-more-documents-list.md`
- `templates/management-summary.md`

## 风险控制

- 缺少监理/发包人确认的已完工程量时，不输出确定应付金额。
- 未确认变更签证只列入争议或暂估，不计入确定付款。
- 支付比例、预付款扣回和质保金必须引用合同条款。

## 人工确认点

- 现场形象进度真实性。
- 未确认变更是否允许暂付。
- 甲供材扣回口径和付款节点。
