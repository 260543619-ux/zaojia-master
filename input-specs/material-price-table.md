# 材料价格表输入规范

## 支持文件格式

- 信息价 PDF/Excel、材料价格表、询价单、供应商报价、采购合同、发票、进场验收记录
- 材料调差计算表、材料台账、甲供材对账表

## 必提字段

| 字段 | 说明 |
|---|---|
| 材料名称 | 与合同清单、信息价名称匹配 |
| 规格型号 | 强度等级、型号、品牌、材质、厚度、直径等 |
| 单位 | t、kg、m3、m、m2、套、台等 |
| 数量 | 合同量、实际用量、调差用量 |
| 基准期价格 | 投标截止日前或合同约定基准期信息价 |
| 施工期价格 | 进场期、使用期或合同约定期别价格 |
| 风险幅度 | 合同约定承包人承担幅度 |
| 调差方法 | 价格信息法、价格指数法、市场询价法、综合系数法 |
| 来源 | 信息价期别、供应商名称、报价日期、发票或合同编号 |

## 可选字段

- 税率、是否含税、运杂费、采保费、损耗率、甲供材扣回
- 施工月份权重、分批进场数量、暂估价确认资料

## 异常判断规则

- 基准期或施工期不清，不能计算确定调差金额。
- 材料名称一致但规格型号不同，应标记为“需人工匹配”。
- 只提供全国参考价或互联网报价时，只能做快速参考和异常提示；如需进入结算、报价或调差计算，应提示用户上传当地信息价、询价、采购合同或发票。
- 未扣除合同约定风险幅度，直接全额调差。
- 调差数量超过合同计量、实际进场或结算确认量。
- 含税/不含税口径不一致，未说明税率和计税方式。

## 缺资料时的追问

- 请提供合同材料调差条款和可调材料清单。
- 请提供基准期、施工期信息价或有效询价记录。
- 请提供材料实际用量、进场验收记录或结算确认量。

## 禁止直接推断

- 不得仅使用内置参考价格生成最终投标价、结算价或审计审减金额。
- 不得凭材料名称近似就自动匹配规格。
- 不得在没有风险幅度条款时自行设定 5% 或 10%。
