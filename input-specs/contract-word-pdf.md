# 合同/招标文件 Word/PDF 输入规范

## 支持文件格式

- `.docx`、`.doc`、`.pdf`、文本化合同条款
- 招标文件、投标人须知、合同协议书、通用条款、专用条款、答疑澄清、补充协议

## 必提字段

| 字段 | 说明 |
|---|---|
| 文件类型 | 合同、招标文件、答疑、补充协议、评标办法 |
| 条款位置 | 页码、章节号、条款号或标题 |
| 合同类型 | 单价合同、总价合同、成本加酬金、EPC 等 |
| 计价依据 | 适用标准、定额版本、信息价、税率方式 |
| 工程范围 | 承包范围、甲供材、暂估价、专业分包界面 |
| 付款条款 | 预付款、进度款、过程结算、竣工结算、质保金 |
| 调价条款 | 材料调差、工程量变化、法律变化、风险幅度 |
| 变更签证 | 指令、申报、审批、计价、时效要求 |
| 争议条款 | 协商、争议评审、仲裁、诉讼 |

## 可选字段

- 最高投标限价、评标基准价计算、清标程序、报价澄清规则
- 违约金、工期奖惩、索赔程序、审计条款
- 发包人代表、监理、造价咨询、授权签字人

## 异常判断规则

- 合同类型不明确，或工程量风险、价格风险分配不清。
- 出现“所有风险由承包人承担”等无限风险表达。
- 材料调差、工程量变化、法律变化没有调整机制。
- 付款比例、审核时限、质保金扣留明显缺少上限或退还条件。
- 变更签证只要求执行，未明确计价、申报、确认和争议处理。
- 招标文件与合同条款冲突，或答疑澄清改变合同核心条件。
- 评标办法未明确报价澄清、算术修正、异常低价或不平衡报价处理。

## 缺资料时的追问

- 请提供合同专用条款和补充协议，不能只看通用条款。
- 请提供招标文件和答疑澄清，以确认合同条款来源。
- 请确认项目资金性质和是否采用 GB/T 50500-2024。
- 请提供可调材料范围和信息价基准期。

## 禁止直接推断

- 不得把商务风险写成法律结论。
- 不得在缺少合同专用条款时认定时效、付款或调价一定适用。
- 不得替代法务意见或评标委员会判断。
