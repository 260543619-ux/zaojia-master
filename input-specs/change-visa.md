# 变更签证输入规范

## 支持文件格式

- 工程变更单、现场签证单、设计变更通知、工程联系单
- 会议纪要、监理指令、照片、视频截图、计算书、报价单

## 必提字段

| 字段 | 说明 |
|---|---|
| 编号 | 变更编号、签证编号、联系单编号 |
| 事件日期 | 发生日期、申报日期、确认日期 |
| 事项描述 | 原因、范围、部位、工作内容 |
| 关联依据 | 合同条款、图纸编号、清单项、会议纪要 |
| 工程量 | 数量、单位、计算式、草图或测量记录 |
| 单价 | 合同已有价、类似价、新组价、询价 |
| 金额 | 申报金额、审核金额、争议金额 |
| 签署 | 发包人、监理、承包人、设计等签字盖章 |
| 附件 | 照片、图纸、计算书、指令、验收记录 |

## 可选字段

- 是否影响工期、关键线路分析、窝工记录、机械停置记录
- 是否属于新增工程、工程变更、现场签证或索赔

## 异常判断规则

- 事项描述不清，仅写“按实计”但无数量和计算依据。
- 签字人授权不明，或缺少合同约定的确认主体。
- 签证内容可能已包含在原合同清单或措施项目中。
- 缺少书面指令、图纸变更或现场影像证据。
- 时效要求需按合同约定核验；缺合同不得直接认定失权。
- 单价未按合同已有价、类似价、新组价顺序说明。

## 缺资料时的追问

- 请提供合同变更签证条款和授权签字人约定。
- 请提供对应图纸、清单项、计算书和现场照片。
- 请说明该事项是否已进入进度款或结算。

## 禁止直接推断

- 不得在缺少合同条款时认定签证无效或丧失权利。
- 不得凭描述直接确认工程量。
- 不得把现场照片作为唯一计量依据。
