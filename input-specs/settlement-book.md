# 结算书输入规范

## 支持文件格式

- 竣工结算书 PDF/Word
- 结算汇总表、单位工程结算表、分部分项结算表、措施项目表、其他项目表、增值税表
- 变更签证汇总表、材料价差表、索赔表、进度款支付表、甲供材对账表

## 必提字段

| 字段 | 说明 |
|---|---|
| 结算报审金额 | 承包人报送总价 |
| 合同价 | 中标价或合同协议书价 |
| 已确认变更 | 变更编号、金额、确认日期 |
| 签证金额 | 签证编号、事项、金额、附件 |
| 材料价差 | 材料名称、基准价、施工期价、用量、调差金额 |
| 已付款 | 预付款、进度款、扣回、已支付合计 |
| 质保金 | 扣留比例、计算基数、扣留金额 |
| 税金 | 计税方式、税率、计税基数 |
| 审核依据 | 合同条款、清单项、定额/信息价、签证附件 |

## 可选字段

- 审定金额、争议金额、暂列金额使用明细、暂估价确认资料
- 甲供材扣回、工期奖罚、索赔和反索赔
- 单位工程、楼栋、专业、成本科目

## 异常判断规则

- 结算价与合同价、变更、签证、材差、索赔、付款逻辑不闭合。
- 变更签证金额进入结算但缺少编号、签字、计算书或附件。
- 合同清单已有内容又通过签证重复计取。
- 材料调差未区分基准期、施工期、风险幅度和合同约定范围。
- 预付款、进度款、甲供材、质保金未按合同扣回。
- 2024 版项目将规费作为独立费用列示，或增值税并入综合单价。

## 缺资料时的追问

- 请提供合同价款条款、已标价工程量清单和结算编制说明。
- 请提供变更签证原件或台账、材料调差依据、付款凭证。
- 请说明结算书是报审版、审核版还是定案版。

## 禁止直接推断

- 不得在缺少合同和签证附件时直接给审减结论。
- 不得用“正常审减率”替代具体审查。
- 不得把疑似问题写成最终审定金额。
