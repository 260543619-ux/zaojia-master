# Excel 清单/报价表输入规范

## 支持文件格式

- `.xlsx`、`.xls`、`.csv`
- 广联达、鲁班、斯维尔等软件导出的清单计价表或结算表
- 手工整理的工程量清单、综合单价分析表、报价对比表

## 必提字段

| 字段 | 说明 |
|---|---|
| 表名/工作表名 | 判断是分部分项、措施、其他项目、税金、汇总表还是综合单价分析表 |
| 项目编码 | 清单项目 12 位编码；无编码时标记为缺失 |
| 项目名称 | 清单项名称 |
| 项目特征 | 规格、型号、材质、做法、强度等级、厚度、部位等 |
| 单位 | m、m2、m3、t、kg、项、个、套等 |
| 工程量 | 数量及小数精度 |
| 综合单价 | 2024 版默认不含增值税 |
| 合价 | 用于校验工程量 x 综合单价 |
| 费用分类 | 分部分项、措施项目、其他项目、增值税 |
| 依据字段 | 定额编号、信息价期别、询价来源、备注 |

## 可选字段

- 人工费、材料费、机械费、企业管理费、利润、风险费
- 主材单价、设备单价、暂估价标识
- 专业、楼栋、部位、清单层级、合同清单编号
- 投标价、控制价、结算价、审定价

## 异常判断规则

- 项目编码为空、位数异常、同一单位工程内重复编码。
- 项目特征为空，或缺少影响组价的强度等级、材质、规格、厚度、施工部位。
- 单位与项目名称明显不匹配，如混凝土用 m2、钢筋用 m3。
- 合价不等于工程量 x 综合单价，且差异超过四舍五入误差。
- GCCP/计价软件导出表中清单工程量、定额工程量、主材量混在同一列或字段来源不清。
- 清单工程量与定额工程量差异较大但没有计算规则说明。
- 2024 版项目中把“规费”作为独立费用项列示。
- 安全生产措施费或增值税被下浮、优惠或并入综合单价。
- 同一清单项在合同清单、变更、签证、结算中重复计取。
- 综合单价显著偏离控制价、投标价、同类项或市场资料，但只能标记为“疑似异常”。

## 缺资料时的追问

- 请提供工程所在地和适用定额/费用文件版本。
- 请提供合同类型、适用标准版本和税率适用方式。
- 请提供清单对应的招标文件、合同清单或结算说明。
- 请说明当前表格是投标报价、最高投标限价、预算书还是结算书。
- 如为广联达/鲁班/斯维尔导出，请同时提供项目设置截图、工作表说明或字段含义说明。

## 禁止直接推断

- 不得凭空补项目编码、定额编号、材料信息价或地方费率。
- 不得仅凭参考指标认定综合单价“错误”，只能提示需核验。
- 不得在没有图纸或做法说明时认定“完整漏项”。
