# 示例：结算审核

## 用户提示词

```text
请按结算审核工作流审核我上传的资料：
1. 施工合同 PDF
2. 竣工结算书 Excel
3. 变更签证台账 Excel
4. 材料调差表 Excel

请先判断资料是否完整，再输出审减/审增问题台账、需补资料清单和管理层摘要。所有问题必须引用文件名、页码或表格行号。
```

## 期望输出结构

1. 资料完整性检查
2. 需补资料清单
3. 结算审核报告摘要
4. 审减/审增问题台账
5. 人工确认点

## 验收要点

- 不在缺合同条款时直接给最终审定金额。
- 对签证和材差问题标注证据来源。
- 对证据不足项使用“疑似”或“待补资料”。
