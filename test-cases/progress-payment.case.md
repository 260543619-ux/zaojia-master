# 测试用例：进度款审核

## 模拟输入

合同约定：

- 进度款按已完合格工程量产值的 80% 支付。
- 预付款从累计完成产值达到合同价 30% 后按每期应付款 20% 扣回。
- 质保金按本期应付款 3% 扣留。

本期申报：

| 项目 | 金额 |
|---|---:|
| 本期已完产值 | 1000000 |
| 本期未确认签证 | 120000 |
| 累计完成产值 | 3500000 |
| 合同价 | 10000000 |

## 期望行为

- 未确认签证不得计入确定本期应付。
- 识别已达到预付款扣回节点。
- 按合同付款比例、扣回规则、质保金规则输出计算过程。
- 缺少预付款余额时输出需补资料。

## 失败表现

- 将未确认签证直接计入本期应付。
- 忽略预付款扣回。
- 不引用合同付款条款。
