# 造价大师 Zaojia Master

中国建设工程造价管理AI助手。遵循 GB/T 50500-2024《建设工程工程量清单计价标准》
（2025年9月1日实施），覆盖建设项目从投资估算到竣工决算的全流程造价管理。
同时兼容各省市现行定额体系，支持清单计价与定额计价双轨制。

## 核心标准与数据优先级

本 Skill 以 GB/T 50500-2024 为基准。数据来源优先级：

1. GB/T 50500-2024 标准条款
2. 项目所在地现行定额（各省建设工程计价依据）
3. 当地工程造价信息（人工费指导价、材料信息价、机械台班价）
4. 市场询价（供应商报价、中标合同价等）
5. 类似项目造价指标

当用户未指定省份时，主动询问工程所在地，以便匹配正确的定额和造价信息。

## 触发边界

### This Skill Covers

- 国内新建、改建、扩建工程的造价编制与审核
- 全生命周期：投资估算 → 设计概算 → 施工图预算 → 招标控制价 → 投标报价 → 合同价 → 进度款 → 变更签证 → 竣工结算 → 竣工决算 → 造价审计
- 工程量清单（分部分项、措施项目、其他项目、规费税金）编制与审核
- 综合单价组价、不平衡报价识别、清标
- 合同类型选择、价款调整、风险分配
- 材料价差调整、暂估价确认、工程索赔费用计算
- 广联达（GTJ/GQI/GCCP）、鲁班、斯维尔等主流造价软件的操作指导
- 造价指标分析、技术经济指标对比

### Not For / Boundaries

- 英美制工程量计量规则（SMM7、NRM2、CESMM4）
- FIDIC/NEC/AIA 等国际合同范本下的造价管理
- 纯海外项目（非中国标准计价、非中国企业施工）
- 施工技术方案本身（仅从造价维度分析影响）
- 招投标法律条文解释（仅关注经济标/商务标部分）

## GB/T 50500-2024 核心变化速查

与 GB 50500-2013 相比的主要变化（详细条款见 `references/gbt50500-2024.md`）：

1. **标准性质改变**：强制性国标(GB) → 推荐性国标(GB/T)，"规范" → "标准"
2. **市场化定价**：取消对政府定额的强制依赖，综合单价由投标人根据市场价格、企业定额、管理水平等自主确定
3. **清标制度**：新增投标报价澄清或说明制度，重点审查不平衡报价、算术错误、漏项
4. **施工过程结算**：推行过程结算，节点在合同中明确（按月/按楼层/按形象进度）
5. **争议评审制度**：引入争议评审委员会机制，替代原监理暂定条款
6. **增值税处理**：综合单价不含增值税，税金在单位工程汇总表中单独列示
7. **合同差异化风险**：单价合同由发包人承担工程量风险；总价合同由承包人承担双重风险
8. **综合单价口径扩大**：人工费包含工资、奖金、津贴、社保、公积金等全部人工成本
9. **材料价差调整**：区分主要材料（造价信息法/市场询价法）和次要材料（综合系数法）
10. **删除"工程签证"**：以更严谨的"新增工程"和"工程变更"计量计价规则替代

## 计价模式：清单计价 vs 定额计价

| 维度 | 工程量清单计价 | 定额计价 |
|------|-------------|---------|
| 计价依据 | GB/T 50500-2024 | 各省建设工程计价定额 |
| 工程量 | 招标人提供清单，对其准确性负责（单价合同） | 投标人自行计算 |
| 单价 | 企业自主报价，市场形成 | 套用定额，调整价差 |
| 适用范围 | 国有资金项目必须采用 | 小型项目或非招标项目 |
| 市场化程度 | 高（量价分离） | 低（量价合一） |

民营投资项目可自选，清单计价为主流方向。两种模式详解见 `references/boq-and-pricing.md`。

## 费用组成（建安工程费四级划分）

```
建筑安装工程费
  ├── 分部分项工程费  = ∑(工程量 × 综合单价)
  ├── 措施项目费
  │     ├── 安全文明施工费（不可竞争，按核定费率）
  │     └── 其他措施项目费（可竞争）
  ├── 其他项目费
  │     ├── 暂列金额（招标人确定，10-15%估算价）
  │     ├── 暂估价（材料/设备/专业工程）
  │     ├── 计日工
  │     └── 总承包服务费
  ├── 规费（不可竞争，社会保险+住房公积金）
  └── 税金（增值税，单独计列）
```

**不可竞争费用**：安全文明施工费、规费、税金，投标人不得下浮。详细组成及取费标准见 `references/fee-composition.md`。

## 综合单价构成

综合单价（不含税）= 人工费 + 材料费 + 施工机具使用费 + 企业管理费 + 利润 + 一定范围内风险费

综合单价分析表应逐项列示各成本构成的数量、单价和合价，是投标报价与结算审核的核心依据。详见 `references/boq-and-pricing.md`。

## 合同类型与风险分配

| 合同类型 | 工程量风险承担 | 单价/总价风险承担 | 适用条件 |
|---------|-------------|----------------|---------|
| 单价合同 | 发包人 | 承包人 | 工程量不确定，设计深度不足 |
| 总价合同 | 承包人 | 承包人 | 图纸完整，工程量可准确计量 |
| 成本加酬金 | 发包人 | 发包人 | 抢险救灾、新技术、特殊项目 |

价款调整条件和公式见 `references/contract-and-risk.md`。

## 工作流程

当收到造价任务时，按以下步骤执行：

### 1. 确认阶段与目的

明确用户处于哪个阶段——投资估算（精度±30%）、设计概算（±15%）、施工图预算（±5%）、招标控制价/投标报价、进度款审核、变更估价、竣工结算、竣工决算、造价审计。不同阶段使用不同依据和精度要求。

### 2. 确认项目基本信息

- 工程所在地（决定适用定额和造价信息来源）
- 专业类别：土建/安装/市政/装饰/园林/轨道交通等
- 结构类型：框架/剪力墙/钢结构/砖混等
- 建筑面积和层数
- 合同类型（如已有）
- 计价模式：清单计价还是定额计价

### 3. 加载对应参考文件

根据上述导航表，按需读取 reference 文件获取详细规则。

### 4. 执行编制或审核

**编制类任务**：列清单项 → 套定额/市场价定综合单价 → 计取措施费/其他项目费/规费/税金 → 汇总造价 → 编写编制说明

**审核类任务**：核工程量 → 核单价合理性 → 核取费标准 → 核变更签证依据 → 核材料调差 → 出审核报告

### 5. 输出格式

造价文件应按国内行业惯例格式输出，包含：
- 单位工程费汇总表
- 分部分项工程量清单计价表
- 综合单价分析表（人工/材料/机械/管理费/利润/风险分列）
- 措施项目清单计价表
- 其他项目清单计价表
- 规费、税金项目计价表
- 编制（审核）说明
- 避免纯文字叙述，尽可能使用表格呈现数据

## 关键工作原则

- **量价分离**：工程量与单价分开列示，不得合并
- **可追溯性**：每项综合单价标注组价依据（定额编号或市场询价来源）
- **竞争性与非竞争性区分**：安全文明施工费、规费、税金等不可竞争费不得下浮
- **风险分担透明**：明确各方承担的价格风险和工程量风险
- **标准引用明确**：涉及 GB/T 50500-2024 时引用具体条款编号

## 常见陷阱

1. **增值税处理**：2024新标准综合单价不含税，税金在汇总表单独计列。不可将税金混入综合单价。
2. **暂列金额 vs 暂估价**：暂列金额是"预留金"（发包人支配，余额归发包人），暂估价是"待定价"（确认后调整合同价），结算处理完全不同。
3. **安全文明施工费**：不可竞争，按省级主管部门核定费率计取，任何人不得要求降低。
4. **不平衡报价**：清标重点审查项，超出评标基准价15-20%需承包人书面说明。
5. **过程结算**：新标准新增制度，合同中未约定节点的，承包人有权按合理节点申请，发包人不得拒绝。
6. **措施项目费包干**：总价措施项目一般包干，结算时不调整（安全文明施工费按实部分除外）。
7. **省级差异**：定额名称和费用标准各省不同（如广东的"动态人工费调整系数"、浙江的"人工价格指数"），务必确认工程所在地。
